# /code-review

Reviews your code and suggests improvements. Accepts an optional effort level argument to control review depth.

```
/code-review
/code-review high
```

Renamed from `/simplify` in v2.1.146.

## 関連リリース

### v2.1.146

- Renamed `/simplify` to `/code-review` with an optional effort level (e.g. `/code-review high`)

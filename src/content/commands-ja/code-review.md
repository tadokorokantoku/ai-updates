# /code-review

コードをレビューして改善案を提示する。オプションでエフォートレベルの引数を指定してレビューの深さを調整できる。

```
/code-review
/code-review high
```

v2.1.146 で `/simplify` から名称変更された。

## 関連リリース

### v2.1.146

- `/simplify` を `/code-review` に名称変更し、オプションでエフォートレベルを指定できるようになった（例：`/code-review high`）

# /login

会話に thinking ブロックが含まれる場合に `/login` 実行後に発生する可能性のある 400 エラーを修正

## 関連リリース

### v2.1.108

- `/login` コードプロンプトでペーストが動作しない問題を修正（2.1.105 のリグレッション）
### v2.1.101

- `/login` OAuth URL がマウス選択をきれいにできないパディング付きでレンダリングされる問題を修正
### v2.1.69

- 複数の OAuth MCP サーバーを使う際の macOS キーチェーン破損を修正。大きな OAuth メタデータ blob が `security -i` の stdin バッファをオーバーフローさせ、古い認証情報が静かに残って `/login` プロンプトが繰り返される可能性があった
### v2.1.30

- 会話に thinking ブロックが含まれる場合に `/login` 実行後に発生する可能性のある 400 エラーを修正

- ユーザー設定のスラッシュコマンドが二重に読み込まれ、レンダリング問題を引き起こすバグを修正
- コマンド説明においてユーザー設定とプロジェクト設定のラベル付けが誤っていた問題を修正

- diff モーダルを閉じる ESC でモデルも中断されてしまう問題を修正
- ctrl-r 履歴検索でスラッシュコマンドにランディングした際に検索がキャンセルされなくなった
- SDK: フックのカスタムタイムアウトをサポート
- より多くの安全な git コマンドを承認なしで実行可能に
- Plugins: output styles の共有とインストールをサポート
- Web からのセッションテレポートで upstream ブランチを自動設定

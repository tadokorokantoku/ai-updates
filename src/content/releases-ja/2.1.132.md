- Bash ツールのサブプロセス環境に `CLAUDE_CODE_SESSION_ID` 環境変数を追加（hooks に渡される `session_id` と一致）
- `CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1` 環境変数を追加 — フルスクリーンの alternate-screen レンダラーを無効化し、会話をターミナルのネイティブスクロールバックに保持
- Ctrl+V での画像貼り付け読み込み中に「Pasting…」フッターヒントを表示するよう追加
- 外部 SIGINT（IDE の停止ボタン、`kill -INT` など）でグレースフルシャットダウンが実行されない問題を修正 — ターミナルモードが復元され、突然の終了の代わりに `--resume` ヒントが表示されるようになった
- ネイティブビルドでセッション中にターミナルが閉じられたり SSH が切断された際に発生する未捕捉例外を修正
- ツールエラーの切り詰めが絵文字を分割した場合に `--resume` が `no low surrogate in string` で失敗する問題を修正。破損したセッションはロード時にサニタイズされるようになった
- `-p --continue`/`--resume` でプランモードセッションを再開する際に `--permission-mode` フラグが無視される問題と、同一セッション内で `ExitPlanMode` 後にプランモードが再適用されない問題を修正
- ノートPCのスリープ/復帰や Ctrl+Z/`fg` 後に次のキー入力やストリーム出力があるまでフルスクリーンモードが空白になる問題を修正
- Indic 合字や ZWJ 絵文字が行をまたぐ場合に Ctrl+E/A/K/U/矢印キーでカーソルが字素の途中に着地する問題を修正
- 分解された (NFD) アクセント文字を含むテキストで vim オペレータがテキストを破損する問題を修正
- `/` で始まるテキストを貼り付けると入力が黙って消えたり不明コマンドの返答が出たりする問題を修正
- フォーカスイベントやマウストラッキングレポートがブラケットペーストに割り込む際に余分なエスケープシーケンスがプロンプトに混入する問題を修正
- upstream の xterm.js バグにより Cursor と VS Code 1.92–1.104 でマウスホイールスクロールが速すぎる問題を修正
- JetBrains IDE 2025.2 ターミナルでのスクロールホイール処理（余分な矢印キー、逆方向イベント、暴走加速）を修正
- Linux/X11 でクリップボードへの統計スクリーンショットコピー時に `/usage` の Ctrl+S がハングする問題を修正
- Windows Terminal で `/terminal-setup` が矛盾するエラーを表示する問題を修正 — そこでは Shift+Enter がネイティブサポートされている
- `CLAUDE_CODE_EFFORT_LEVEL` 環境変数のオーバーライドが `/effort` ピッカーに反映されない問題を修正
- 一部ユーザーで `/status` が間違ったデフォルトモデルを表示する問題を修正
- スラッシュコマンドの自動補完ポップアップが ~3〜5 コマンドで上限になりターミナル高さに応じてスケールしない問題を修正
- statusline の `context_window` トークン数がセッション累計ではなく現在のコンテキスト使用量を反映するよう修正
- macOS ターミナルで「Option as Meta」が有効になっていない場合（iTerm2、Terminal.app のデフォルト）に Alt+T（thinking トグル）が動作しない問題を修正
- `claude agents` からバックグラウンドセッションを再度開いた後に Windows でキーボード入力が無反応になる問題を修正
- stdio MCP サーバーが非プロトコルデータを stdout に書き込む際のメモリ使用量の無制限な増大（10GB+ RSS）を修正
- `tools/list` への接続に失敗した MCP サーバーが静かに 0 ツールと表示される問題を修正 — 1 回リトライし、`/mcp` に「connected · tools fetch failed」と表示されるようになった
- 未認証の claude.ai MCP コネクターが「needs auth」ではなく「failed」と表示される問題と、ヘッドレス `-p` モードで一時的でない 4xx 接続エラーをリトライする問題を修正
- スラッシュコマンドダイアログと `/login`、`/upgrade`、`/extra-usage` ダイアログのスペーシングの視覚的一貫性を改善
- `/tui fullscreen` 起動バナーを更新し、追加レンダラーのメリット（低メモリ使用量、マウスサポート、選択時の自動コピー）を説明するようにした
- `ENABLE_PROMPT_CACHING_1H` 設定時の Bedrock と Vertex の 400 エラーを修正

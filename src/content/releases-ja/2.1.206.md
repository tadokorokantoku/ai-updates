- `/cd` に `/add-dir` と同様のディレクトリパス候補表示機能を追加
- コードベースから推測できる内容を削って、checked-in の `CLAUDE.md` ファイルの簡素化を提案する `/doctor` のチェック項目を追加
- `/commit-push-pr` が、`origin` に加えてリポジトリの設定済み push リモート（`remote.pushDefault`、または設定されているリモートが1つのみの場合はそのリモート）への `git push` を自動許可するようになった
- Gateway: `/login` が Anthropic 運用のパブリック gateway エンドポイントに対応
- `EnterWorktree` が、プロジェクトの `.claude/worktrees/` ディレクトリ外の git worktree に入る前に確認を求めるようになった
- バックグラウンドエージェントが、Claude Code のアップデート直後にバックグラウンドで新バージョンにアップグレードされるようになり、アタッチ時に発生していた遅いストールセッションのアップグレードが不要に
- ログインの期限切れが、`/login` の実行を促す代わりに、すべてのモデルで「There's an issue with the selected model」という誤解を招くエラーとして表示されていた問題を修正
- `claude --resume` と `--continue` が起動時にキーボード入力に反応しない問題を修正
- `--mcp-config` または `.mcp.json` で設定した MCP サーバーがサーバーごとの `request_timeout_ms` を無視し、新規セッションで実行時間の長い MCP ツール呼び出しがデフォルトの60秒でタイムアウトしていた問題を修正
- `CLAUDE_CODE_EXTRA_BODY` が `claude agents` / `--bg` のバックグラウンドワーカーで黙って無視されていた問題を修正。シェルでエクスポートされたオーバーライドがディスパッチ元のセッションに引き継がれるようになった
- OAuth の MCP サーバーが、トークンのリフレッシュに一度失敗しただけで手動の再認証を要求していた問題を修正
- `--permission-prompt-tool` が MCP サーバーを指している場合、サーバーの接続完了前のコールドスタート時に「MCP tool not found」でクラッシュしていた問題を修正
- `/model` ピッカーの行が、行の名前と異なるモデルの価格を表示していた問題と、料金を請求しないプロバイダーでファーストパーティのリスト価格を誤って表示していた問題を修正
- entitlement やアローリストの制限で行が配置されるはずだった位置の行がドロップされた場合に、サーバー提供のモデル行が `/model` ピッカー内で誤った位置に表示される問題を修正
- スラッシュコマンドがターンの途中で送信された後、デスクトップセッションが「running」表示のまま固まる問題を修正
- Windows で `claude --resume` を単体実行した際、セットアップ用のプロンプトが先に表示されるとエージェントビューでキーボード入力が無視される問題を修正
- `claude rm` が削除したジョブをデーモンの管理台帳に残してしまい、`claude agents` にその行が再表示される問題を修正
- ログアウト状態で `/remote-control` を実行すると「Unknown command」と表示されていた問題を修正。サインイン方法を案内するようになった
- ワークフロー詳細ビューで左矢印キーがフェーズやエージェントから一段階戻れない問題を修正
- `/status` に同じ「broken-install」警告が二重に表示される問題を修正
- LSP プラグインに対する誤った「disused plugin」の表示と、不正確な disuse テレメトリを修正
- `/doctor` の更新チェックが、Homebrew インストールを settings のチャンネルではなく cask のチャンネルと比較するように修正
- フルスクリーンの「最下部へジャンプ」ピルが、macOS で Ctrl+End を提案していた問題、リバインドされたコードを表示していなかった問題、トランスクリプトの上に折り返してしまっていた問題を修正
- Bedrock: egress が制限されたネットワークで `awsCredentialExport` ヘルパーを使用した際に発生していた数分単位の起動ハングを修正
- claude-opus-4-8 での `/code-review` の指摘品質をすべてのエフォートレベルで改善
- エージェントビューの改善: ステータス列が64文字での切り詰めではなく、ターミナルの全幅を使用するようになった
- エージェントビューの変更: Ctrl+X が完了したセッションを完全に削除するようになり、セッションが二重に表示されなくなった。削除したバックグラウンドジョブは削除済みのまま維持される

- ターン時間メッセージ（例: "Cooked for 1m 6s"）を非表示にする `showTurnDuration` 設定を追加
- 権限プロンプトを承認する際にフィードバックを提供する機能を追加
- タスク通知内でエージェントの最終応答をインライン表示。完全なトランスクリプトファイルを読まずに結果を確認しやすく
- ワイルドカード権限ルールがシェル演算子を含む複合コマンドにマッチするセキュリティ脆弱性を修正
- クラウド同期ツール、アンチウイルススキャナ、Git がコンテンツを変更せずにファイルタイムスタンプを触れた場合に Windows で発生する誤った「file modified」エラーを修正
- ストリーミング実行中に兄弟ツールが失敗したときの孤立した tool_result エラーを修正
- コンテキストウィンドウブロック制限が実効コンテキストウィンドウ（max output tokens のスペースを予約）ではなく完全なコンテキストウィンドウを使用して計算される問題を修正
- `/model` や `/theme` などのローカルスラッシュコマンドを実行する際にスピナーが一瞬点滅する問題を修正
- 固定幅の braille 文字を使用してターミナルタイトルアニメーションのジッターを修正
- git サブモジュールを持つプラグインがインストール時に完全に初期化されない問題を修正
- temp ディレクトリパスが `t` や `n` のような文字をエスケープシーケンスとして誤解釈されると Windows で bash コマンドが失敗する問題を修正
- ターミナルレンダリングのメモリ割り当てオーバーヘッドを削減してタイピング応答性を改善
- すべてのユーザー向けに MCP ツール検索 auto モードをデフォルトで有効化。MCP ツール記述がコンテキストウィンドウの 10% を超えると、自動的に遅延して事前ロードではなく MCPSearch ツールで発見されるように。多くの MCP ツールを設定しているユーザーのコンテキスト使用量を削減。設定で `disallowedTools` に `MCPSearch` を追加することで無効化可能
- OAuth と API Console の URL を console.anthropic.com から platform.claude.com に変更
- [VSCode] `claudeProcessWrapper` 設定が Claude バイナリパスではなく wrapper パスを渡す問題を修正

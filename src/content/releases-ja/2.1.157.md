- `.claude/skills` ディレクトリのプラグインが自動的に読み込まれるようになり、マーケットプレイス不要に
- `.claude/skills` に新しいプラグインをスキャフォールドする `claude plugin init <name>` を追加
- `/plugin` 引数のオートコンプリートを追加（サブコマンド、インストール済みプラグイン名、既知のマーケットプレイスのプラグイン）
- `claude agents`：`settings.json` の `agent` フィールドがディスパッチセッションで適用されるようになり、`--agent <name>` でオーバーライド可能に
- `EnterWorktree` がセッション中に Claude 管理の worktree 間を切り替えられるようになった
- `OTEL_LOG_TOOL_DETAILS=1` 設定時に `tool_decision` テレメトリイベントへ `tool_parameters`（bash コマンド、MCP/スキル名）が含まれるようになった
- Claude が管理する worktree はエージェント終了後にロック解除されるようになり、`git worktree remove`/`prune` でクリーンアップ可能に
- 貼り付け・MCP・ダイアログ経由で添付した処理不能な画像（ゼロバイト、破損）がリクエストをクラッシュさせる代わりにテキストプレースホルダーになるよう修正
- デスクトップアプリ・IDE 拡張機能・SDK 使用時の auto および bypass-permissions モードでサンドボックスネットワーク許可プロンプトが表示される問題を修正
- アイドル状態のサブエージェントが残存またはバックグラウンドシェルがリークしている場合に `claude agents` の完了セッションが終了しない問題を修正
- `claude agents` で Esc を押しても遅い「opening…」がキャンセルされずリストが応答しなくなる問題を修正
- 30 日間のジョブ保持期間後に `.claude/worktrees/` 配下のバックグラウンドエージェント worktree が孤立する問題を修正
- スリープ/ウェイク後に再アタッチしたバックグラウンドセッションがモデルに正しい日付を伝えない問題を修正
- `set-clipboard on` の tmux 内で `claude agents` のコピー選択がシステムクリップボードに届かない問題を修正（2.1.153 からのリグレッション）
- `--resume` が前回の Claude Code プロセス終了時に実行中だったバックグラウンドサブエージェントを報告しない問題を修正
- `--resume` のセッションピッカーがフルスクリーンモードで終了後も内容を端末に残す問題を修正
- `--worktree` および `--worktree --tmux` が現在リンクされた worktree でなく正規のリポジトリルートに戻る問題を修正
- 選択中のモデルが既にファミリー内の最新であっても `/model` ピッカーが誤った「新しいバージョンが利用可能」ヒントを表示する問題を修正。ピン留めモデル行にはモデルの生 ID ではなく説明が表示されるように
- フルスクリーンモードの進行中メッセージテキストにリテラルのマークダウン記号（バッククォート、アスタリスク）が表示される問題を修正
- 起動時のマネージド設定セキュリティダイアログを承認した後に端末がフリーズする問題を修正
- 端末 UI の再描画後にスクロールバックに稀に重複行が表示される問題を修正
- VS Code・Cursor・Windsurf の統合ターミナルで右クリック貼り付けがクリップボードを複製する問題を修正
- WSL：画像貼り付け（`alt+v` キーバインド）と Windows 11 でのスクリーンショット貼り付けを修正し、Windows Explorer からの画像ドラッグをサポート
- 冗長なメッセージレンダリング再計算を排除して長い会話や再開した会話のパフォーマンスを改善
- `/terminal-setup` が VS Code/Cursor/Windsurf の統合ターミナルで文字化けレンダリングを防ぐために GPU アクセラレーションを無効化するようになった
- Feature of the Week のクレジット請求ステータスがプロンプトの上の行ではなくステータスエリアの通知として表示されるようになった
- `claude agents`：ディスパッチ入力のスラッシュコマンドオートコンプリートが部分文字列一致するようになった
- 「bash コマンドはサンドボックス化されます」起動バナーを削除（サンドボックスのステータスは `/status` とコマンドがブロックされた際に引き続き表示）
- 「/ide for …」起動ヒントトーストを削除
- [IDE] バックグラウンドサブエージェント実行中に Stop をクリックしても実際に停止しない問題を修正
- [VSCode] Opus 4.8 で fast モードインジケーターが表示されない問題を修正
- ワークフロートリガーキーワードの直後に Backspace を押すとワークフローリクエストが `alt+w` と同様に解除されるようになった（文字削除の代わりに）
- プロンプト内の「workflow」という単語が動的ワークフローをトリガーしないようにするための「Workflow keyword trigger」設定を `/config` に追加

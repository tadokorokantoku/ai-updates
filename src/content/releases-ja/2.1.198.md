- Claude in Chrome が正式リリース (GA)
- `claude agents` にバックグラウンドエージェントの通知を追加 — 入力待ちや完了時に `Notification` フック (`agent_needs_input` / `agent_completed`) が発火するように
- チャート・ダッシュボード設計ガイダンスを提供する `/dataviz` スキルを追加。実行可能なカラーパレットバリデータ付き
- Gateway: アップストリームプロバイダとして Claude Platform on AWS (anthropicAws) を追加。model-not-found レスポンスでフェイルオーバーチェーンが進むように
- `claude agents` から起動したバックグラウンドエージェントが、worktree でのコード作業完了時に確認を求めて停止する代わりに、コミット・プッシュしてドラフト PR を自動で作成するように
- 組み込みの Explore エージェントが haiku ではなく、メインセッションのモデル (opus を上限) を引き継ぐように
- サブエージェントとコンテキスト圧縮がセッションの拡張思考設定を引き継ぐようになり、委任タスクの出力品質が向上
- 応答中の一時的なネットワーク切断でターンが中断される問題を修正 — ECONNRESET などの一時的なエラーは失敗する代わりにバックオフ付きでリトライされるように
- サンドボックス化されたプロセスが同じネットワークホストに繰り返しアクセスした際の、過剰なバックグラウンド分類リクエストを修正
- web・デスクトップ・VS Code のタスクパネルで、バックグラウンドタスクが完了後やセッション再開後に「実行中」のまま止まる問題を修正
- agent teams を修正: API エラーで停止したチームメイトがリードに「failed」を報告するように。また、スタックしたチームメイトにメッセージを送るとすぐに再試行するように起こされるように
- ブランチを切り替えたりセッション外でコミットした際に `/diff` パネルが更新されない問題を修正
- フルスクリーンモードで Markdown テーブルがはみ出し、右端がラップされる問題を修正
- STS トークンの期限切れ時に Claude Platform on AWS と Mantle のセッションが「Please run /login」で行き詰まる問題を修正 — `awsAuthRefresh` が自動的に実行されるように
- macOS のバックグラウンドエージェントセッションで、ローカルネットワークの entitlements を宣言することでローカルホストへの「no route to host」を修正
- worktree に出入りした後、`/desktop` が「Cannot determine working directory」で失敗する問題を修正
- agents ビューを開いたままにしていると、macOS でバックグラウンドエージェントが約52秒ごとに「Reconnecting…」を繰り返し表示する問題を修正
- `claude attach <id>` 内で `←` を押すとエージェントビューを開く代わりにシェルに抜けてしまう問題を修正
- `claude --bg` を `--print`/`-p` と併用した際に、アタッチ不能なセッションが無言で作成される問題を修正。矛盾するフラグは事前に拒否されるように
- SDK およびデスクトップアプリのセッションで、フェーズカウンタは正しいままワークフロー進捗ビューから最も古いエージェントがリストから抜け落ちる問題を修正
- シンボリックリンク経由でファイルに到達した場合に `.claude/rules/` の条件付きルールが読み込まれない問題を修正
- macOS の Warp のフルスクリーンモードで Cmd+クリックが URL を開かない問題を修正
- フルスクリーンモードでのダブルクリックによる単語選択が、スキームを含む URL 全体を選択するように修正
- セッションが plan mode で開始した際に、読み取り専用ツール呼び出しが自動許可されない問題を修正
- `/branch` のデフォルトのフォーク名が、最初の実プロンプトではなく圧縮サマリーから導出されていた問題を修正
- フォーカスモードを改善: ターン内で起動されたサブエージェントがアクティビティサマリーに表示されるように。完了したバックグラウンド通知は1つのカウントにまとめられるように
- highlight.js 11 へのアップグレードにより、コードブロック・diff・ファイルプレビューの構文ハイライト精度が向上
- Mac から SSH 接続時に、キーボードショートカットのヒントが alt/super ではなく opt/cmd で表示されるように
- API リトライ UX を改善: 2回目の試行後にエラー理由が表示されるように。API が過負荷の場合、スピナーのヒントの代わりにステータスページへのリンクが表示されるように
- `/login` が「利用できません」と表示する代わりに、`claude agents` ビューからサインインダイアログを開くように
- サブエージェントが、起動元のエージェントからのメッセージを通常のタスク指示として扱うように。エージェントからのメッセージがユーザーの承認として扱われることは引き続きない
- `/agents` ウィザードを削除。サブエージェントの作成・管理は Claude に依頼するか、`.claude/agents/` を直接編集すること

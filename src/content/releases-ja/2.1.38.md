- 2.1.37 で入った VS Code ターミナルの scroll-to-top リグレッションを修正
- Tab キーがオートコンプリートではなくスラッシュコマンドをキューに入れる不具合を修正
- 環境変数ラッパーを使うコマンドの bash 権限マッチングを修正
- ストリーミング不使用時にツール使用間のテキストが消える不具合を修正
- VS Code 拡張で再開時にセッションが重複する不具合を修正
- コマンド密輸を防ぐためヒアドキュメント区切りパースを改善
- sandbox モードで `.claude/skills` ディレクトリへの書き込みをブロック

- MCP `headersHelper` スクリプトに `CLAUDE_CODE_MCP_SERVER_NAME` と `CLAUDE_CODE_MCP_SERVER_URL` 環境変数を追加。1 つのヘルパーが複数のサーバーを提供可能に
- 権限ルール構文（例: `Bash(git *)`）を使うフックの条件付き `if` フィールドを追加。実行タイミングをフィルタリングしてプロセス spawn オーバーヘッドを削減
- スケジュールされたタスク（`/loop`、`CronCreate`）が発火するときのトランスクリプトのタイムスタンプマーカーを追加
- 画像を貼り付けるときに `[Image #N]` プレースホルダーの後に末尾スペースを追加
- ディープリンククエリ（`claude-cli://open?q=…`）が 5,000 文字までサポート、長い事前入力プロンプトには「scroll to review」警告
- MCP OAuth が authorization サーバーを見つけるために RFC 9728 Protected Resource Metadata discovery に従うように
- 組織ポリシー（`managed-settings.json`）によってブロックされたプラグインはインストールまたは有効化できず、マーケットプレイスビューから非表示に
- `permissionDecision: "allow"` と並行して `updatedInput` を返すことで PreToolUse フックが `AskUserQuestion` を満たすように。独自の UI で回答を収集するヘッドレス統合を可能に
- OpenTelemetry の tool_result イベントの `tool_parameters` が `OTEL_LOG_TOOL_DETAILS=1` の背後でゲートされるように
- 会話が compact リクエスト自体に収まらないほど大きくなった場合に `/compact` が "context exceeded" で失敗する問題を修正
- プラグインのインストール場所が settings で宣言された場所と異なる場合、`/plugin enable` と `/plugin disable` が失敗する問題を修正
- 非 git リポジトリで `--worktree` が `WorktreeCreate` フックが実行される前にエラーで終了する問題を修正
- `deniedMcpServers` 設定が claude.ai MCP サーバーをブロックしない問題を修正
- マルチモニターセットアップで computer-use ツールの `switch_display` が "not available in this session" を返す問題を修正
- `OTEL_LOGS_EXPORTER`、`OTEL_METRICS_EXPORTER`、`OTEL_TRACES_EXPORTER` が `none` に設定されたときのクラッシュを修正
- 非ネイティブビルドで diff シンタックスハイライトが動作しない問題を修正
- refresh トークンが存在するときに MCP step-up authorization が失敗する問題を修正 — `403 insufficient_scope` 経由で elevated スコープを要求するサーバーが正しく再認証フローをトリガーするように
- ストリーミング応答が中断されたときのリモートセッションのメモリリークを修正
- エッジ接続の churn 中の持続的な ECONNRESET エラーを、リトライ時に新鮮な TCP 接続を使用して修正
- 特定のスラッシュコマンドを実行した後にプロンプトがキューでスタックし、up-arrow で取得できない問題を修正
- Python Agent SDK: `--mcp-config` 経由で渡された `type:'sdk'` MCP サーバーが起動時にドロップされない問題を修正
- SSH 経由または VS Code 統合ターミナルで実行中に生のキーシーケンスがプロンプトに表示される問題を修正
- 権限が解決された後も Remote Control セッションステータスが "Requires Action" でスタックする問題を修正
- shift+enter と meta+enter が改行を挿入する代わりに typeahead 候補によってインターセプトされる問題を修正
- ストリーミング中に上にスクロールすると古いコンテンツが透けて見える問題を修正
- Ghostty、Kitty、WezTerm、その他の Kitty キーボードプロトコルをサポートするターミナルで終了後にターミナルが拡張キーボードモードのままになる問題を修正 — 終了後に Ctrl+C と Ctrl+D が正しく動作するように
- 大きなリポジトリでの @ メンションファイル autocomplete パフォーマンスを改善
- PowerShell ツールの危険なコマンド検出を改善
- WASM yoga-layout を純粋な TypeScript 実装に置き換えて大きなトランスクリプトでのスクロールパフォーマンスを改善
- 大きなセッションでコンパクションがトリガーされたときの UI スタッターを削減

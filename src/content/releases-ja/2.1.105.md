- 現在のリポジトリの既存ワークツリーに切り替える `path` パラメータを `EnterWorktree` ツールに追加
- PreCompact フックサポートを追加: フックは終了コード 2 または `{"decision":"block"}` を返すことで compaction をブロック可能
- プラグイン用のバックグラウンドモニターサポートを、セッション開始時またはスキル呼び出し時に自動アーム化されるトップレベルの `monitors` マニフェストキー経由で追加
- `/proactive` を `/loop` のエイリアスに
- 停滞した API ストリーム処理を改善: ストリームが 5 分間データなしで無期限にハングするのではなく、非ストリーミングにリトライして中止
- ネットワークエラーメッセージを改善: 接続エラーは無音のスピナーではなく即座にリトライメッセージを表示
- ファイル書き込み表示を改善: 長い単一行の書き込み（例: minify された JSON）は多数の画面にページングするのではなく UI で切り詰め
- `/doctor` のレイアウトをステータスアイコン付きで改善。`f` を押すと Claude が報告された問題を修正
- 明確さのために `/config` のラベルと説明を改善
- スキルの説明文ハンドリングを改善: 一覧表示の上限を 250 から 1,536 文字に引き上げ、説明が切り詰められたときに起動時の警告を追加
- `WebFetch` を改善し、取得したページから `<style>` と `<script>` の内容を除去。CSS の多いページが実際のテキストに到達する前にコンテンツ予算を使い切らないように
- PR が squash-merge されたワークツリーを無期限に保持するのではなく削除するように、古いエージェントワークツリーのクリーンアップを改善
- MCP の大出力切り詰めプロンプトを改善し、フォーマット固有のレシピ（例: JSON には `jq`、テキストには計算された Read チャンクサイズ）を提供
- Claude が作業中に送信されたキューのメッセージに添付された画像がドロップされる問題を修正
- 長い会話でプロンプト入力が 2 行目に折り返されるときに画面が空白になる問題を修正
- 全画面モードで複数行のアシスタント応答を選択するときに先頭の空白がコピーされる問題を修正
- アシスタントメッセージから先頭の空白がトリムされ、ASCII アートやインデントされた図が壊れる問題を修正
- コマンドがクリック可能なファイルリンクを出力するとき（例: Python の `rich`/`loguru` ロギング）の bash 出力が文字化けする問題を修正
- ESC プレフィックス alt エンコーディングを使うターミナルで alt+enter が改行を挿入しない問題と、Ctrl+J が改行を挿入しない問題を修正（2.1.100 のリグレッション）
- EnterWorktree/ExitWorktree ツール表示で「Creating worktree」のテキストが重複する問題を修正
- focus モードでキューのユーザープロンプトが消える問題を修正
- ファイルウォッチャーが発火後のクリーンアップを見逃したときに、ワンショットのスケジュールタスクが繰り返し再発火する問題を修正
- Team/Enterprise ユーザー向けに、最初のメッセージの後に inbound チャンネル通知が静かにドロップされる問題を修正
- `package.json` とロックファイルを持つマーケットプレイスプラグインが、インストール/更新後に依存関係が自動インストールされない問題を修正
- プラグインプロセスが更新中にファイルを開いたままにしていると、公式マーケットプレイスが壊れた状態になるマーケットプレイス自動更新の問題を修正
- `/resume`、`--worktree`、`/branch` の後の終了時に「Resume this session with...」ヒントが印刷されない問題を修正
- フィードバックアンケートのショートカットキーが、長いプロンプトの末尾で入力したときに発火する問題を修正
- stdio MCP サーバーが不正な（非 JSON）出力を出したときに、「Connection closed」で早期失敗せずセッションがハングする問題を修正
- MCP サーバーが非同期で接続する場合、headless/remote-trigger セッションの最初のターンで MCP ツールが欠落する問題を修正
- AWS Bedrock の非 US リージョンで inference profile の発見が進行中の間に `/model` ピッカーが無効な `us.*` モデル ID を `settings.json` に永続化する問題を修正
- API キー、Bedrock、Vertex ユーザーで 429 レート制限エラーが、クリーンなメッセージではなく生の JSON ダンプを表示する問題を修正
- セッションに不正なテキストブロックが含まれているときの再開時クラッシュを修正
- 短いターミナル高さで `/help` がタブバー、Shortcuts 見出し、フッターを表示しない問題を修正
- `keybindings.json` の不正なキーバインドエントリ値が、明確なエラーで拒否されずに静かにロードされる問題を修正
- あるプロジェクトの設定の `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC` が、マシン上のすべてのプロジェクトで使用状況メトリクスを永続的に無効化する問題を修正
- SSH/mosh 経由で Ghostty、Kitty、Alacritty、WezTerm、foot、rio、Contour を使うときに 16 色パレットが薄く表示される問題を修正
- plan モードを終了するとより高い権限レベルから降格する場合に、Bash ツールが `acceptEdits` 権限モードを提案する問題を修正

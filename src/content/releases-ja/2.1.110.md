version: "2.1.110"
date: 2026-04-15T22:07:12+00:00
highlights:
  - "レンダリングモードを切り替える `/tui` スラッシュコマンドを追加"
  - "フォーカスビューを切り替える `/focus` スラッシュコマンドを追加"
---

- `/tui` コマンドと `tui` 設定を追加 — `/tui fullscreen` で同じ会話内でちらつきのないレンダリングに切り替え可能
- プッシュ通知ツールを追加 — Remote Control と "Push when Claude decides" 設定が有効な場合、Claude がモバイルプッシュ通知を送信可能
- `Ctrl+O` を通常表示と verbose トランスクリプトの切り替えのみに変更。フォーカスビューは新しい `/focus` コマンドで個別に切り替え
- フルスクリーンモードで会話の自動スクロールを無効化する `autoScrollEnabled` 設定を追加
- `Ctrl+G` 外部エディタで Claude の最後のレスポンスをコメント付きコンテキストとして表示するオプションを追加 (`/config` で有効化)
- `/plugin` Installed タブを改善 — 注意が必要な項目とお気に入りが先頭に表示、無効項目は折りたたみ内に非表示、`f` で選択項目をお気に入り登録
- 複数のコンフィグスコープで異なるエンドポイントの MCP サーバーが定義されている場合に警告するよう `/doctor` を改善
- `--resume`/`--continue` が期限切れでないスケジュールタスクを復元するように
- `/context`、`/exit`、`/reload-plugins` が Remote Control (モバイル/ウェブ) クライアントから動作するように
- IDE diff で提案内容を編集してから受け入れた場合、Write tool がモデルに通知するように
- Bash tool が任意の大きな値を受け入れず、ドキュメント記載の最大タイムアウトを強制するように
- SDK/ヘッドレスセッションが分散トレース連携のため環境から `TRACEPARENT`/`TRACESTATE` を読み込むように
- テレメトリが無効のユーザー (Bedrock、Vertex、Foundry、`DISABLE_TELEMETRY`) でセッション recap を有効化。`/config` または `CLAUDE_CODE_ENABLE_AWAY_SUMMARY=0` でオプトアウト。
- SSE/HTTP トランスポートでレスポンス途中にサーバー接続が切断された場合に MCP tool 呼び出しが無期限にハングする問題を修正
- API が到達不能な場合に非ストリーミングフォールバックのリトライが数分間のハングを引き起こす問題を修正
- フォーカスモードでセッション recap、ローカルスラッシュコマンド出力、その他のシステムステータス行が表示されない問題を修正
- ツール実行中にテキストが選択されているときにフルスクリーンで高 CPU 使用率になる問題を修正
- マーケットプレイスエントリが省略している場合に、プラグインインストールが `plugin.json` で宣言された依存関係を尊重しない問題を修正。`/plugin` install が自動インストールされた依存関係を表示するように
- `disable-model-invocation: true` のスキルがメッセージ途中で `/<skill>` 経由で呼び出された際に失敗する問題を修正
- 実行中または異常終了したセッションで `--resume` が `/rename` 名の代わりに最初のプロンプトを表示することがある問題を修正
- マルチツール呼び出しターン中にキューに入ったメッセージが一時的に二重表示される問題を修正
- サブエージェントトランスクリプトを含むセッションディレクトリ全体がセッションクリーンアップで削除されない問題を修正
- CLI 再起動後 (例: `/tui`、プロバイダセットアップウィザード) にキーストロークがドロップされる問題を修正
- macOS Terminal.app など同期出力をサポートしないターミナルで起動時のレンダリングが崩れる問題を修正
- 信頼できないファイル名からのコマンドインジェクションに対して "Open in editor" アクションを強化
- `updatedInput` を返す `PermissionRequest` フックが `permissions.deny` ルールに対して再チェックされない問題を修正。`setMode:'bypassPermissions'` の更新は `disableBypassPermissionsMode` を尊重するように
- ツール呼び出し失敗時に `PreToolUse` フックの `additionalContext` がドロップされる問題を修正
- stdout に JSON 以外の行を出力する stdio MCP サーバーが最初の迷子行で切断される問題を修正 (2.1.105 のリグレッション)
- `CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC` または `CLAUDE_CODE_DISABLE_TERMINAL_TITLE` が設定されている場合、ヘッドレス/SDK セッションの自動タイトル生成が余分な Haiku リクエストを発行する問題を修正
- パイプ (非 TTY) Ink 出力が非常に幅広の 1 行を含む場合に過剰なメモリ割り当てが発生する可能性を修正
- フルスクリーンモードでリストがモーダルをオーバーフローしたときに `/skills` メニューがスクロールしない問題を修正
- Remote Control セッションが古すぎる場合に再ログインを促す代わりに汎用エラーを表示する問題を修正
- claude.ai からの Remote Control セッションのリネームがローカル CLI セッションにタイトルとして永続化されない問題を修正

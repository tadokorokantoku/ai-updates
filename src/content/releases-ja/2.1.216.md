- ファイルシステムの隔離をスキップしつつネットワーク egress 制御は維持できる `sandbox.filesystem.disabled` 設定を追加
- 長時間セッションでメッセージ正規化のコストがターン数に対して二次関数的に増大し、数秒単位の停止や resume の遅延を引き起こしていた不具合を修正
- OAuth トークンがセッション途中で期限切れ・ローテーションした後、auto mode が「HTTP 401」の classifier エラーを含むコマンドを拒否していた不具合を修正
- AskUserQuestion が、回答が「待って」「説明して」という趣旨だった場合でも Claude に続行するよう伝えていた不具合を修正 — 自由記述の回答には中立的な文言が使われるように
- Claude Code on the web で、セッションが数分間アイドル状態だった後に同じ質問を繰り返し尋ね、回答を取りこぼしていた不具合を修正
- ファイル変更を伴う hook 実行後に @-mention が何も添付しなかった問題、vim の `c` 系オペレータやペーストの dot-repeat の問題、resume 時に statusline が二重実行される問題、resume ピッカーが失敗時にハングする問題を修正
- resume したバックグラウンドエージェントセッションがデフォルトエージェントに戻ってしまう不具合を修正: エージェントの prompt と tool 制限が復元されるように
- worktree で隔離されたサブエージェントが `git -C`、`--git-dir`、`GIT_DIR`/`GIT_WORK_TREE` を介して共有チェックアウトへ git 操作をリダイレクトしてしまう不具合を修正
- 作業ディレクトリが選択中のプロジェクトと一致しない場合、worktree セッションが別プロジェクトの残存 worktree に着地してしまう不具合を修正
- worktree に git リポジトリが存在しないバックグラウンドセッションが削除不能になる不具合を修正
- `claude daemon stop --any` が、古いレガシー daemon lockfile により無関係なプロセスを終了させてしまう可能性があった不具合を修正
- 長時間実行中でバックグラウンドタスクがあるセッションで、アイドル状態のプロンプトから Esc-Esc しても rewind ピッカーが開かない不具合を修正
- `&&` リストや否定の中にリダイレクトを含む複合 Bash コマンドの permission チェックを修正
- エージェント一覧で Ctrl+X を2回押してもセッションを削除できない不具合、バックグラウンドワーカーが停止した際に削除済みセッションが再表示される不具合を修正
- 起動中のバックグラウンドサブエージェントが、起動処理中に高優先度メッセージが届くとキャンセルされてしまう不具合を修正
- `/memory`、`/plan`、`/keybindings`、Ctrl+G から GUI エディタを開いている間、ターミナルにマウスやフォーカスのゴミが残る不具合を修正。`/memory` はエディタが閉じるのを待たなくなった
- セッションの OAuth トークンに必要なスコープがない場合に Claude-in-Chrome が再接続時に 403 ループしていた不具合を修正
- workflow の保存や scheduled task の書き込みが `.claude` のシンボリックリンクをたどり、プロジェクト外への書き込みにつながる可能性があった不具合を修正
- MCP の再認証が、新しいサインインが成功する前に有効な認証情報を失効させていた不具合、およびバックグラウンドセッションでの再接続の「認証が必要」メッセージが使用不能なコマンドを指していた不具合を修正
- Windows で読み取り専用コマンドが permission プロンプトなしにネットワークパスへアクセスできてしまう不具合を修正
- 実際のシェルの単語境界に一致するよう、非 ASCII 文字を含む Bash コマンドのパース処理を修正
- 見えない Unicode 文字を含むコマンドに対する PowerShell tool の permission 検証を修正
- フルスクリーンモードでダイアログがパネル右端を超えて伸びてしまう不具合を修正
- フルスクリーンモードの `/config` 設定一覧で、キーボードヒントのフッターが見切れる不具合を修正
- 104 カラムより狭いターミナルで transcript モード（Ctrl+O）のフッターヒントが折り返されてしまう不具合を修正
- Prometheus メトリクスエンドポイント（`OTEL_METRICS_EXPORTER=prometheus`）が不正な `# UNIT` 行を出力していた不具合を修正
- セッション中に変更された skill や command が、再起動するまでスラッシュメニューに反映されない不具合を修正
- `name` frontmatter フィールドを持つ plugin skill が、スラッシュコマンドの自動補完で plugin プレフィックスを失っていた不具合を修正
- permission 拒否に関する telemetry の誤報告を修正: permission プロンプトのリクエスト失敗がユーザーの拒否としてカウントされなくなり、ユーザーによる中断は拒否ではなく abort として報告されるように
- `/fork` の確認メッセージを、新しいセッション名・`claude attach` id・チェックアウトを共有する場合の注記を含む1行に改善
- PowerShell tool における `git` と `gh` コマンド引数の検証を改善
- `/ultrareview` の diff too large エラーで、設定されている上限・実際の diff サイズ・寄与の大きいファイルを表示するように改善
- `/code-review ultra` の空 diff メッセージで、対象の base ref を明示し明示的な base の指定を提案するように改善
- spend limit の変更が拒否された場合に、サーバー側の理由を表示するよう spend limit 調整プロンプトを改善
- `/context` が会話がコンテキストウィンドウを超えた場合に明示的な警告を表示するようになり、`/compact` の失敗はエラーとして表示されるように
- `/rewind` が追跡対象パスのシンボリックリンクやハードリンク経由でファイルを復元・削除しなくなり、スキップしたパス数を報告するように
- バックグラウンドセッション: クライアントが接続されていない場合、`/mcp` と `/install-github-app` がエージェントビューに「入力待ち」リクエストを表示するように
- 同梱の dataviz skill を更新: デフォルトのチャートパレットの並び順を修正し、4系列チャートで direct label を推奨していたガイダンスを修正
- [VSCode] 右から左に書く言語（アラビア語、ヘブライ語、ペルシャ語）が英語やコードと混在した際に表示順が崩れる不具合を修正
- ターン中にセッションのコンテナが再起動した際にクラウドセッションが処理中のメッセージを取りこぼしていた不具合を修正 — 中断されたターンはセッションが応答不能になる代わりに resume 時に再実行されるように

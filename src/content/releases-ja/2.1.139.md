- agent ビュー（Research Preview）を追加: 実行中・返答待ち・完了済みのすべての Claude Code セッションを一覧表示。`claude agents` で開始。詳細: https://code.claude.com/docs/en/agent-view
- `/goal` コマンドを追加: 完了条件を設定するとターンをまたいで Claude が達成まで作業を継続。インタラクティブ・`-p`・Remote Control で動作し、経過時間/ターン数/トークン数をオーバーレイパネルでリアルタイム表示
- `/scroll-speed` コマンドを追加: マウスホイールのスクロール速度をライブプレビュー付きで調整
- `claude plugin details <name>` を追加: プラグインのコンポーネント一覧とセッションあたりの推定トークンコストを表示
- トランスクリプトビューのナビゲーションを追加: `?` でキーボードショートカット表示、`{`/`}` でユーザープロンプト間のジャンプ、`v` でショートカットパネルの切り替え
- hook の `args: string[]` フィールド（exec 形式）を追加: シェルを介さず直接コマンドを起動するため、パスのプレースホルダーをクォートする必要がない
- `PostToolUse` 用の hook `continueOnBlock` 設定オプションを追加: `true` に設定すると hook の拒否理由を Claude にフィードバックしてターンを継続
- MCP stdio サーバーが hook と同様に環境変数 `CLAUDE_PROJECT_DIR` を受け取るように。プラグイン設定のコマンドで `${CLAUDE_PROJECT_DIR}` を参照可能
- compaction プロンプトがモデルに対してユーザーの重要な指示を保持するよう要求するように
- `/mcp` の Reconnect が再起動なしで `.mcp.json` の編集を反映、再接続失敗時に HTTP ステータスと URL を表示
- `/context all` のスキルごとのトークン推定値がモデルのトークナイザーを考慮し丸めた値を表示
- `claude plugin install <name>@<marketplace>` がプラグインが見つからない場合に marketplace を自動更新してリトライするように
- `/plugin` のインストール済みプラグイン詳細に hook イベント名と MCP サーバー名をわかりやすく表示
- `/context` がプラグイン提供のスキルに提供元プラグイン名を表示
- Remote MCP サーバーの一時的な障害時の再接続リトライをすべてのユーザーに有効化
- サブエージェントからの API リクエストに `x-claude-code-agent-id` / `x-claude-code-parent-agent-id` ヘッダーを付与、`claude_code.llm_request` OTEL スパンに `agent_id` / `parent_agent_id` 属性を追加
- `ANTHROPIC_API_KEY` / `apiKeyHelper` / `ANTHROPIC_AUTH_TOKEN` が設定されている場合、Claude.ai ログインが存在しても Remote Control・`/schedule`・claude.ai MCP コネクター・通知設定を無効化。これらの機能を使用するには API キーを削除
- 期限切れの認証情報と `forceRemoteSettingsRefresh` ポリシー設定が `claude auth login`/`logout`/`status` をデッドロックして復旧不能になる問題を修正
- `$VAR` や `$(cmd)` のようなシェル展開を含むコマンドを `autoAllowBashIfSandboxed` が自動承認しない問題を修正
- hook がターミナルに書き込むと画面上のインタラクティブプロンプトが破損するバグを修正: hook はターミナルアクセスなしで動作するように
- HTTP/SSE MCP サーバーが非プロトコルデータをストリーミングした際のメモリ無限増大を修正: SSE フレームあたり 16 MB に上限を設定
- `Skill(name *)` の権限ルールを修正: ワイルドカード形式が `Bash(ls *)` の動作に合わせてプレフィックスマッチとして機能するように
- シンボリックリンクされた `~/.claude/settings.json` の編集を設定のホットリロードが検出しない問題を修正
- マーケットプレイスのキーがマニフェスト名と異なる場合にプラグイン詳細の読み込みが失敗する問題を修正
- `/model` ピッカーの "Default" 行が `ANTHROPIC_DEFAULT_OPUS_MODEL`/`ANTHROPIC_DEFAULT_SONNET_MODEL` の上書き設定を反映しない問題を修正
- ストリームキャンセル時にウォッチドッグタイマーがクリアされないことが原因で、レスポンス完了から 5 分後に誤って "stream idle timeout" が発生する問題を修正
- 10 台以上の MCP サーバーが設定されてキャッシュディレクトリが書き込み不可の場合に `exit 1` がサイレントで発生する問題を修正: エラーメッセージに根本原因を含めるように
- タブ名・リストポインター・ダイアログの選択行にカーソルが点滅する問題を修正

- Bash モードで上矢印キーの履歴が最初のエントリを繰り返し、編集中のドラフトを上書きする問題を修正

- ダークテーマでハイパーリンクに読みにくい濃紺が使用される問題を修正: アクティブなテーマに合わせて適応するように

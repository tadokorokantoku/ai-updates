- ログインの有効期限が近づくと警告を表示するようになり、バックグラウンドセッションが中断される前に再認証できるようにした
- manual permission mode の際にフッターへ灰色の ⏸ バッジを表示し、現在のモードを常に確認できるようにした
- セッションの追加作業ディレクトリを MCP の `roots/list` に含めるようにし、その集合が変化した際は `notifications/roots/list_changed` を送信するようにした
- macOS でバックグラウンドエージェントセッションを開く/切り替える際に、誤った低メモリ検出が原因で 15〜20 秒固まる問題を修正（2.1.196 で発生したリグレッション）
- デーモンのセッショントークンが古くなった際に、バックグラウンドセッションへの attach・返信・停止が永久に応答しなくなる問題を修正 — セッションは自動的に復旧するようになった
- `claude agents` に戻ると実行中のサブエージェントが黙って停止し、プロンプトが最初からやり直しになる問題を修正 — 作業内容が引き継がれるようになった
- インタラクティブセッションでのメモリおよび毎ターンの CPU リグレッションを修正：コンテキスト使用量インジケーターが毎ターン transcript 全体を再解析しなくなった
- バックグラウンドエージェントが、ディスパッチ元のシェルではなくデーモンの古い `PATH` を継承してしまい、Windows でツールが見つからなくなる問題を修正
- バックグラウンドおよび agent-view セッションが、シェルからエクスポートされた `ANTHROPIC_BASE_URL` を失ってしまい、API キーがデフォルトのエンドポイントに送られて 401 で失敗する問題を修正
- git worktree が多数あるリポジトリで Bash が "argument list too long" で失敗する問題を修正
- worktree で分離されたサブエージェントが、自分の worktree ではなく親のチェックアウトでシェルコマンドを実行してしまうことがある問題を修正
- マルチリポジトリのワークスペースで worktree の作成がネストしたリポジトリを拒否し、バックグラウンドセッションが分離・編集できなくなる問題を修正
- 作業ディレクトリが削除された、ファイルに置き換わった、または不正なパスになった際に、バックグラウンドエージェントがクラッシュループする問題を修正 — 現在は一度だけ失敗し、明確なエラーを表示する
- バックグラウンドデーモンの自動アップグレード失敗が、実行中のバックグラウンドセッションをすべて黙って強制終了させてしまう問題を修正
- 別のエージェントが起動したバックグラウンドエージェントを `TaskStop` と `TaskOutput` が見つけられない問題を修正 — エラーには実行中のエージェントが id と説明付きで一覧表示されるようになった
- `claude agents` のコンポーザーで、スラッシュコマンドが利用できない場面で入力中のメッセージが破棄される問題を修正
- 別のセッションで既に開かれている会話を持つ停止済みセッションを開くとエージェント一覧がクラッシュする問題を修正
- バックグラウンドセッションが、質問に既に回答済みであるにもかかわらずエージェント一覧で「Needs input」と表示され続ける問題を修正
- バックグラウンドエージェントの起動失敗時に、実際のエラーではなく "exit_with_message" のみが表示される問題を修正
- デーモン経由で fork した際に、バックグラウンドセッションが settings.json の `effortLevel` の変更を無視する問題を修正
- attach されたバックグラウンドセッションが `CLAUDE_CODE_DISABLE_MOUSE` と `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` のオプトアウトを無視する問題を修正
- 名前付きエージェントがすべて完了した後でも `/exit` が実行中のバックグラウンドエージェントについて誤った警告を出す問題を修正
- 非 git ディレクトリから開始したバックグラウンドセッションで、`WorktreeCreate` フックが設定されている場合にファイルを編集できない問題を修正
- `claude agents` の `@` ディレクトリピッカーに登録済みの git worktree が表示されない問題を修正
- Windows で `/clear` 後にバックグラウンドタスクの出力が空ファイルで永続的に置き換わってしまう問題を修正
- 長い transcript 履歴を上方向にスクロールした際にコンテンツがジャンプする問題を修正
- シェル履歴の候補が表示されている際に bash モードで入力するとターミナルがちらついたりジャンプしたりする問題を修正
- バックグラウンドセッションへの再アタッチ時に、`^[[I` / `^[[O` というエスケープコードがそのまま表示される問題を修正
- 言語サーバーが診断やナビゲーション要求に応答しているにもかかわらず、LSP 専用の plugin が未使用として誤って検出される問題を修正
- 長い応答のストリーミング中の応答性を改善：ライブプレビューの更新で画面全体が再描画されなくなった
- サブエージェントの挙動を改善：エージェントがタスク全体を別のサブエージェントに再委任してしまうことが少なくなった
- バンドルされた大きな依存関係をインライン化せず遅延読み込みにすることで、バイナリサイズを約 7MB、起動時メモリを約 7MB 削減
- 左矢印キーでバックグラウンドタスク、diff、workflow の詳細ビューが閉じないように変更 — 代わりに Esc を押す
- `claude agents` の空の表示で、説明付きの分類済みセクション（Needs input / Working / Completed）を常に表示するように変更
- 起動時の "claude command missing or broken" 警告を削除 — 代わりに `/doctor` と `/status` に表示されるようになった
- `claude agents` フッターの冗長なナビゲーションヒントを削除
- [VSCode] 「Enable Remote Control for all sessions」の設定トグルを追加

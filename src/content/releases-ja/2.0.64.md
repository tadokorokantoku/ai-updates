- 自動 compact を即時化
- エージェントと bash コマンドが非同期実行され、メインエージェントを起こすメッセージを送信可能に
- /stats でお気に入りモデル、利用グラフ、利用ストリークなど CC の興味深い統計情報を提供
- 名前付きセッションサポートを追加。`/rename` でセッションに名前を付け、REPL で `/resume <name>` やターミナルから `claude --resume <name>` で再開可能
- .claude/rules/ のサポートを追加。詳細は https://code.claude.com/docs/en/memory を参照
- 画像がリサイズされた際の寸法メタデータを追加。大きな画像に対する正確な座標マッピングが可能に
- ネイティブインストーラー使用時の .env 自動読み込みを修正
- `--continue` や `--resume` フラグ使用時に `--system-prompt` が無視される問題を修正
- `/resume` 画面を改善し、フォークされたセッションをグループ化、プレビュー (P) とリネーム (R) のキーボードショートカットを追加
- VSCode: コードブロックと bash ツール入力にコピー用ボタンを追加
- VSCode: Windows ARM64 で拡張機能が動作しない問題をエミュレーション経由の x64 バイナリへのフォールバックで修正
- Bedrock: トークンカウントの効率を改善
- Bedrock: `aws login` の AWS マネジメントコンソール認証情報のサポートを追加
- AgentOutputTool と BashOutputTool を廃止し、新しい統合 TaskOutputTool に置き換え

- Windows 向け PowerShell ツールをオプトインプレビューとして追加。詳細は https://code.claude.com/docs/en/tools-reference#powershell-tool
- サードパーティ（Bedrock、Vertex、Foundry）向けに pin されたデフォルトモデルの effort/thinking 能力検出をオーバーライドする `ANTHROPIC_DEFAULT_{OPUS,SONNET,HAIKU}_MODEL_SUPPORTS` 環境変数、および `/model` ピッカーラベルをカスタマイズする `_MODEL_NAME`/`_DESCRIPTION` を追加
- ストリーミングアイドルウォッチドッグ閾値を設定する `CLAUDE_STREAM_IDLE_TIMEOUT_MS` 環境変数を追加（デフォルト 90s）
- `TaskCreate` 経由でタスクが作成されたときに発火する `TaskCreated` フックを追加
- `WorktreeCreate` フックが `type: "http"` をサポート — レスポンス JSON の `hookSpecificOutput.worktreePath` で作成された worktree パスを返す
- team/enterprise 管理者がチャンネルプラグインの allowlist を定義する `allowedChannelPlugins` 管理対象設定を追加
- タイムアウトのデバッグ用に API リクエストに `x-client-request-id` ヘッダーを追加
- 75 分以上後に戻ってきたユーザーに `/clear` を促すアイドル復帰プロンプトを追加。古いセッションでの不要なトークン再キャッシュを削減
- 検出リストの最初にあるターミナルではなく、ディープリンク（`claude-cli://`）が優先ターミナルで開くように
- ルールとスキルの `paths:` frontmatter が YAML の glob リストを受け付けるように
- MCP ツール記述とサーバー指示は 2KB で cap され、OpenAPI 生成サーバーがコンテキストを膨張させるのを防止
- ローカルと claude.ai コネクタの両方で設定された MCP サーバーが重複排除されるように — ローカル設定が優先
- インタラクティブプロンプトでスタックしているように見えるバックグラウンド bash タスクが、約 45 秒後に通知を表示
- 1M 以上のトークン数が "1512.6k" ではなく "1.5m" として表示
- MCP ツールを設定したユーザーを含め、`ToolSearch` 有効時にグローバルシステムプロンプトキャッシングが動作するように
- voice push-to-talk を修正: voice キーを押し続けても文字がテキスト入力にリークせず、トランスクリプトが正しい位置に挿入されるように
- フッターアイテムにフォーカスがあるときに up/down 矢印キーが反応しない問題を修正
- 複数行入力の行境界で `Ctrl+U`（kill-to-line-start）が no-op になる問題を修正。`Ctrl+U` を繰り返すと行をまたいでクリアされるように
- デフォルトのコードバインディング（例: `"ctrl+x ctrl+k": null`）を null にアンバインドしてもコード待機モードに入る問題を修正。プレフィックスキーを解放するように
- マウスイベントがトランスクリプト検索入力にリテラルの "mouse" テキストを挿入する問題を修正
- 外側のセッションが `--json-schema` を使用し、サブエージェントもスキーマを指定するとワークフローサブエージェントが API 400 で失敗する問題を修正
- 一部のターミナルでユーザーメッセージバブル内の特定の絵文字の背景色が欠落する問題を修正
- `Edit(.claude)` allow ルールを持つユーザーで「allow Claude to edit its own settings for this session」権限オプションが維持されない問題を修正
- 大きな編集されたファイル用のアタッチメントスニペットを生成するときのハングを修正
- サーバー再接続時の MCP ツール/リソースキャッシュリークを修正
- 部分クローンリポジトリ（Scalar/GVFS）が大量の blob ダウンロードをトリガーする起動パフォーマンス問題を修正
- ネイティブターミナルカーソルがテキスト入力キャレットを追跡しない問題を修正。IME 合成（CJK 入力）がインラインでレンダリングされ、スクリーンリーダーが入力位置を追跡できるように
- 一時的な keychain 読み取り失敗による macOS での誤った「Not logged in」エラーを修正
- コアツールがバイパスがアクティブなしで遅延される可能性があり、Edit/Write が typed パラメータで InputValidationError で失敗するコールドスタートレースを修正
- Windows ドライブルート（`C:\`、`C:\Windows` など）の危険な削除の検出を改善
- `setup()` をスラッシュコマンドとエージェントのロードと並列実行してインタラクティブ起動を約 30ms 改善
- MCP サーバー付きの `claude "prompt"` の起動を改善 — すべてのサーバーが接続するまでブロックする代わりに REPL が即座にレンダリング
- Remote Control がブロックされたときに generic な「not yet enabled」メッセージの代わりに特定の理由を表示するよう改善
- p90 プロンプトキャッシュ率を改善
- メッセージウィンドウをコンパクションとグルーピング変更から免疫にすることで、長いセッションでのスクロールトップリセットを削減
- アニメーションツール進捗がビューポート上にスクロールしたときのターミナルちらつきを削減
- issue/PR 参照が `owner/repo#123` として書かれた場合のみクリック可能なリンクになるように変更 — bare `#123` はもはや自動リンクされない
- 現在の認証設定で利用できないスラッシュコマンド（`/voice`、`/mobile`、`/chrome`、`/upgrade` など）は表示されず非表示に
- [VSCode] 使用率と reset 時刻を表示するレート制限警告バナーを追加
- Stats スクリーンショット（/stats の Ctrl+S）がすべてのビルドで動作するように、16 倍高速

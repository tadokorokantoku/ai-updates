- MCP elicitation サポートを追加 — MCP サーバーがインタラクティブダイアログ（フォームフィールドまたはブラウザ URL）経由でタスク中に構造化された入力を要求可能に
- 応答を送信する前にインターセプトしてオーバーライドする新しい `Elicitation` と `ElicitationResult` フックを追加
- 起動時にセッションの表示名を設定する `-n` / `--name <name>` CLI フラグを追加
- 大規模なモノレポで `claude --worktree` 用に git sparse-checkout 経由で必要なディレクトリのみをチェックアウトする `worktree.sparsePaths` 設定を追加
- コンパクション完了後に発火する `PostCompact` フックを追加
- モデル effort レベルを設定する `/effort` スラッシュコマンドを追加
- セッション品質調査を追加 — エンタープライズ管理者は `feedbackSurveyRate` 設定でサンプリングレートを設定可能
- 遅延ロードされた（`ToolSearch` 経由の）ツールが会話のコンパクション後に入力スキーマを失い、配列と数値パラメータが型エラーで拒否される問題を修正
- スラッシュコマンドが "Unknown skill" を表示する問題を修正
- プランが既に受諾された後にプランモードが再承認を求める問題を修正
- 権限ダイアログまたはプランエディタが開いているときに voice モードがキー入力を吸収する問題を修正
- npm 経由でインストールされたときの Windows で `/voice` が動作しない問題を修正
- 1M コンテキストセッションで `model:` frontmatter のスキルを呼び出した際に誤った "Context limit reached" が表示される問題を修正
- 非標準モデル文字列を使用した際の "adaptive thinking is not supported on this model" エラーを修正
- 引用された引数に `#` が含まれるときに `Bash(cmd:*)` 権限ルールがマッチしない問題を修正
- パイプと複合コマンドの Bash 権限ダイアログの "don't ask again" が完全な生のコマンドを表示する問題を修正
- 連続した失敗後に自動コンパクションが無限リトライする問題を修正。3 回の試行後に停止するサーキットブレーカーを追加
- 再接続成功後に MCP reconnect スピナーが残る問題を修正
- マーケットプレイスが調整される前に LSP Manager が初期化されると、LSP プラグインがサーバーを登録しない問題を修正
- SSH 経由で tmux のクリップボードコピーを修正 — 直接ターミナル書き込みと tmux クリップボード統合の両方を試行
- 成功メッセージでファイル名のみ表示する代わりに完全なファイルパスを表示するよう `/export` を修正
- テキスト選択後にトランスクリプトが新しいメッセージに自動スクロールしない問題を修正
- ログイン方法選択画面を終了するために Escape キーが動作しない問題を修正
- 複数の Remote Control 問題を修正: サーバーがアイドル環境を reap したときにセッションが無言で死ぬ、ラピッドなメッセージがバッチではなく一度に 1 つずつキューされる、JWT 更新後に古い作業項目が再配信される
- 拡張 WebSocket 切断後にブリッジセッションが回復に失敗する問題を修正
- soft-hidden コマンドの正確な名前を入力するとスラッシュコマンドが見つからない問題を修正
- git refs を直接読み、リモートブランチがローカルで既に利用可能なときに冗長な `git fetch` をスキップして `--worktree` 起動パフォーマンスを改善
- バックグラウンドエージェントの動作を改善 — バックグラウンドエージェントを停止すると、部分的な結果が会話コンテキストに保持されるように
- モデルフォールバック通知を改善 — verbose モードの背後に隠される代わりに常に可視、人間向けモデル名で
- ダークターミナルテーマでのブロッククォートの読みやすさを改善 — dim ではなくイタリックと左バーで表示
- 古い worktree クリーンアップを改善 — 中断された並列実行後に残された worktree が自動クリーンアップされる
- Remote Control セッションタイトルを改善 — "Interactive session" を表示する代わりに最初のプロンプトから派生
- `/voice` を改善。有効化時に dictation 言語を表示し、voice 入力で `language` 設定がサポートされていないときに警告
- サブコマンドをサポートするために `--plugin-dir` が 1 つのパスのみ受け入れるよう更新 — 複数のディレクトリには `--plugin-dir` を繰り返し使用
- [VSCode] カンマを含む gitignore パターンが @ メンションファイルピッカー全体のファイル型を無言で除外する問題を修正

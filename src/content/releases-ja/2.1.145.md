- `claude agents --json` を追加し、スクリプト用途（tmux-resurrect、ステータスバー、セッションピッカーなど）向けにライブ Claude セッションを JSON 形式で一覧表示できるようになった
- `claude_code.tool` OTEL スパンに `agent_id` と `parent_agent_id` 属性を追加し、バックグラウンドサブエージェントのスパンが Agent ツールスパン配下に正しくネストされるようトレース親子関係を修正した
- ステータスライン JSON 入力に、検出された場合の GitHub リポジトリと PR 情報が含まれるようになった
- `/plugin` の Discover・Browse 画面でインストール前にプラグインのコマンド、エージェント、スキル、フック、MCP/LSP サーバーが表示されるようになった
- `claude agents` のターミナルタブタイトルに入力待ちのエージェント数が表示されるようになり、alt-tab で切り替えた別ウィンドウからでもエージェントが入力を待っているかどうかわかるようになった
- フルスクリーンモードでスラッシュコマンドおよび @-mention の候補リストがマウスホバーとクリックに対応した
- Stop および SubagentStop フックの入力に `background_tasks` と `session_crons` フィールドが追加された
- Bash コマンド内でアローリストに登録されていない環境変数への単純な変数代入が自動承認されてしまうパーミッションプロンプトのバイパスを修正した
- MCP プロンプトのスラッシュコマンドで必須引数が省略された際にサーバーの生のバリデーションエラーが表示されていた問題を修正した。エラーメッセージが欠落している引数名と期待される使用方法を示すようになった
- ターミナルのリサイズまたは再フォーカス後にキー入力があるまでスピナーと経過時間の表示が固まってしまう問題を修正した
- デフォルトの Windows PowerShell 5.1 でクロスプロジェクト再開のヒントが失敗していた問題を修正した。Windows のコマンド区切り文字として `;` を使用するようになった
- エージェントビューの返信ペインでボイスプッシュトゥトークが機能しない問題を修正した
- 複数のタスクが同時に作成された場合にタスクリストがランダムな順序でレンダリングされる問題を修正した
- Anthropic マーケットプレイスがすでにインストール済みの場合でも「Failed to install Anthropic marketplace」バナーが表示され続ける問題を修正した
- セッション内で `gh pr create` などの PR 状態変更コマンドを実行した直後にフッターの PR バッジが更新されない問題を修正した
- Agent Teams で非 ASCII 名のチームメンバーが無効なヘッダーエンコードにより API 呼び出しに毎回失敗する問題を修正した
- `/review` が Classic Projects を持つリポジトリでエラーになる非推奨の `projectCards` GraphQL クエリを使用していた問題を修正した
- `claude plugin validate` が `skills:` エントリでディレクトリではなくファイルを指定している場合にフラグを立てない問題を修正した。エラーメッセージに親ディレクトリを指定するよう提案が表示されるようになった
- `context: fork` を使用するスキルが自分自身を繰り返し再呼び出しし続ける無限ループを修正した
- Read ツールがファイル全体の読み込みがトークン制限を超えた場合にハードエラーではなく「PARTIAL view」の通知付きで最初のページの切り捨てた内容を返すよう改善した

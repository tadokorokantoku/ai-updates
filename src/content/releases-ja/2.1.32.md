- Claude Opus 4.6 が利用可能に
- マルチエージェント協調のためのリサーチプレビュー agent teams 機能を追加 (トークン消費が大きい機能。CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS=1 の設定が必要)
- Claude が作業中にメモリを自動的に記録・想起するように
- メッセージセレクタに "Summarize from here" を追加。部分的な会話サマリ化が可能に
- 追加ディレクトリ (`--add-dir`) 内の `.claude/skills/` に定義されたスキルを自動読み込み
- サブディレクトリから実行したときに `@` ファイル補完が誤った相対パスを表示する不具合を修正
- --resume はデフォルトで直前の会話で指定された --agent の値を再利用するよう更新
- 修正: ヒアドキュメントに `${index + 1}` のような JavaScript テンプレートリテラルが含まれるときに Bash ツールが "Bad substitution" エラーを投げてツール実行を中断する不具合
- スキルの文字数予算がコンテキストウィンドウに応じてスケール (コンテキストの 2%) するように。大きなコンテキストウィンドウのユーザーはより多くのスキル説明を切り詰めなしで閲覧可能に
- タイ語/ラオ語の母音記号 (สระ า, ำ) が入力フィールドで正しくレンダリングされない不具合を修正
- VSCode: 入力フィールドに先行テキストがある状態で Enter を押すとスラッシュコマンドが誤って実行される不具合を修正
- VSCode: 過去の会話リスト読み込み中にスピナーを追加

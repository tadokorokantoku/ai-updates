- Chrome 拡張機能 (https://claude.ai/chrome) と連携する Claude in Chrome (Beta) 機能を追加。Claude Code から直接ブラウザを操作可能
- ターミナルのちらつきを低減
- モバイルアプリのダウンロードを簡単にするスキャン可能な QR コードをヒントに追加
- 会話再開時のローディングインジケーターを追加しフィードバックを改善
- 非対話モードで `/context` コマンドがカスタムシステムプロンプトを尊重しない問題を修正
- Ctrl+Y で貼り付けるときの連続する Ctrl+K 行の順序を修正
- @ メンションのファイル候補の速度を改善（git リポジトリで約 3 倍高速化）
- `.ignore` や `.rgignore` ファイルがあるリポジトリでのファイル候補のパフォーマンスを改善
- 設定バリデーションエラーをより目立つように改善
- 誤操作を避けるため thinking トグルを Tab から Alt+T に変更

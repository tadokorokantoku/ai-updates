- `/config` コマンドに設定を素早くフィルタリングする検索機能を追加
- 自動更新チャンネルと利用可能な npm バージョン（stable/latest）を表示する Updates セクションを `/doctor` に追加
- `/stats` コマンドに日付範囲フィルタリングを追加。`r` を押して Last 7 days、Last 30 days、All time を切り替え
- サブディレクトリのファイルで作業する際、ネストされた `.claude/skills` ディレクトリからスキルを自動発見する機能を追加
- コンテキストウィンドウ表示を容易にする `context_window.used_percentage` と `context_window.remaining_percentage` フィールドをステータスライン入力に追加
- Ctrl+G 時にエディタが失敗した場合のエラー表示を追加
- ブロックされたコマンドを実行させる可能性のあるシェル行継続による権限バイパスを修正
- ファイルウォッチャーがコンテンツを変更せずにファイルに触れた際の誤った「File has been unexpectedly modified」エラーを修正
- 複数行の応答でテキストスタイル（太字、色）が徐々にずれる問題を修正
- 説明フィールドに 'n' を入力した際にフィードバックパネルが予期せず閉じる問題を修正
- 週次リセット後の低使用率でレート制限警告が表示される問題を修正（70% 使用率を必要とするように）
- 以前のセッションを再開した際にレート制限オプションメニューが誤って自動的に開く問題を修正
- Kitty キーボードプロトコルターミナルで numpad キーが文字ではなくエスケープシーケンスを出力する問題を修正
- Kitty キーボードプロトコルターミナルで Option+Return が改行を挿入しない問題を修正
- ホームディレクトリに破損した設定バックアップファイルが蓄積する問題を修正（設定ファイルごとに 1 つのバックアップのみ作成）
- `mcp list` と `mcp get` コマンドが孤立した MCP サーバープロセスを残す問題を修正
- `display:none` でノードが非表示になるときの ink2 モードでの視覚的アーティファクトを修正
- 外部 CLAUDE.md インポート承認ダイアログを改善。どのファイルがどこからインポートされるかを表示
- バックグラウンドタスクが 1 つだけ実行されているときにタスク詳細に直接遷移するよう `/tasks` ダイアログを改善
- @ オートコンプリートを改善。候補タイプごとのアイコンと単一行フォーマット
- 「Help improve Claude」設定の取得を更新。OAuth トークンの期限切れで失敗した際、OAuth を更新してリトライ
- 複数のバックグラウンドタスクが同時に完了したときにタスク通知の表示を 3 行で制限し、オーバーフロー要約を表示するように変更
- ウィンドウ識別を改善するため、起動時にターミナルタイトルを "Claude Code" に変更
- MCP サーバーの有効化/無効化のための @ メンションを削除。代わりに `/mcp enable <name>` を使用
- [VSCode] 手動コンパクション後に使用量インジケータが更新されない問題を修正

- 現在のセッションに URL からプラグイン `.zip` アーカイブを取得する `--plugin-url <url>` フラグを追加
- 自動検出が失敗するターミナル（例：Emacs `eat`）で同期出力を強制的に有効化する `CLAUDE_CODE_FORCE_SYNC_OUTPUT=1` 環境変数を追加
- `CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE` を追加：Homebrew または WinGet インストール環境で設定すると、Claude Code がバックグラウンドでアップグレードコマンドを実行し、再起動を促すように
- プラグインマニフェスト：`themes` と `monitors` は `"experimental": { ... }` 配下で宣言するように。トップレベルの宣言は引き続き動作するが、`claude plugin validate` が警告を出すように
- `/model` ピッカー向けの Gateway `/v1/models` ディスカバリーが `CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY=1` によるオプトイン方式に変更（2.1.126〜2.1.128 では自動だった）
- Ctrl+R 履歴ピッカーが全プロジェクトのプロンプトを横断検索するデフォルト動作に戻った（2.1.124 以前の挙動に一致）。現在のプロジェクトまたはセッションに絞り込むには Ctrl+S を押す
- サードパーティデプロイメント（Bedrock、Vertex、Foundry、または `ANTHROPIC_BASE_URL` ゲートウェイ）のスピナーのヒントがファーストパーティ Anthropic の画面を指さなくなった
- `skillOverrides` 設定が機能するように：`off` はモデルと `/` の両方から非表示、`user-invocable-only` はモデルのみから非表示、`name-only` は説明を折りたたむ
- `claude_code.pull_request.count` OTel メトリクスが、シェルコマンドだけでなく MCP ツール経由で作成された PR/MR もカウントするように
- ポリシー拒否エラーメッセージにサポートデバッグを容易にするための API Request ID が含まれるように
- 認識されない 400 ステータスコードの API エラーで、根本的なエラーメッセージの代わりに生の JSON が表示される問題を修正
- 会話後に `/clear` がターミナルのタブタイトルをリセットしない問題を修正
- 権限ダイアログなどが表示されている間に `/rename` によるセッションタイトルチップが消える問題を修正
- サブエージェント実行中にプロンプト下のエージェントパネルが非表示になる問題を修正（2.1.122 のリグレッション）
- 外部エディターへの引き渡し（Ctrl+G）でプロンプト上の会話履歴が消える問題を修正
- `/context` がレンダリングされた ASCII 可視化グリッドを会話に投入し、呼び出しごとに約 1.6k トークンを無駄にしていた問題を修正
- `/agents` ライブラリリストの矢印キーナビゲーションを修正：リストがビューポートを超えても選択中のエージェントが表示され続けるように
- `/branch` の成功メッセージに `/resume` 用の新しいブランチのセッション ID が含まれていなかった問題を修正
- フルスクリーンモードでキーキャップ/ZWJ/スキントーン絵文字を含む太字ヘッダーの末尾の文字が欠ける問題を修正
- 保存された OAuth 認証情報に `user:inference` スコープがないエンタープライズ/チームユーザーにサーバー管理の設定ポリシーが適用されない問題を修正
- スリープ復帰後の OAuth リフレッシュ競合により実行中の全セッションがログアウトされる可能性があった問題を修正
- 1 時間のプロンプトキャッシュ TTL がサイレントに 5 分にダウングレードされていた問題を修正
- `/effort` または `/model` を変更した後に `/clear` またはコンパクション後にキャッシュミス警告が誤って表示される問題を修正
- `Bash(mkdir *)` や `Bash(touch *)` などの allow ルールがプロジェクト内のパスに対して適用されない問題を修正
- `*://` スキームワイルドカードを持つ `deniedMcpServers` パターンが大文字小文字混在のホスト名にマッチしない問題を修正
- ボイスモード中の `--debug` で無害な WebSocket 警告がエラーとして記録される問題を修正
- [VSCode] `/clear` が会話コンテキストと表示されたトランスクリプトをクリアしない問題を修正

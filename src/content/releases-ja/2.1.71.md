version: "2.1.71"
date: 2026-03-07T00:11:59+00:00
highlights:
  - "プロンプトを定期実行する `/loop` コマンドの新規追加"
---

- プロンプトまたはスラッシュコマンドを定期的に実行する `/loop` コマンドを追加（例: `/loop 5m check the deploy`）
- セッション内での定期プロンプト用の cron スケジューリングツールを追加
- voice activation キーを `keybindings.json` で再バインド可能にする `voice:pushToTalk` キーバインドを追加（デフォルト: space）— `meta+k` のような modifier+letter の組み合わせはタイピング干渉がゼロ
- bash 自動承認 allowlist に `fmt`、`comm`、`cmp`、`numfmt`、`expr`、`test`、`printf`、`getconf`, `seq`、`tsort`、`pr` を追加
- キー入力が処理されなくなるがプロセスは生きたままになる長時間実行セッションでの stdin フリーズを修正
- システムウェイク後に CoreAudio 初期化がメインスレッドをブロックすることで voice モード有効ユーザーの 5〜8 秒の起動フリーズを修正
- 多くの claude.ai プロキシコネクタが同時に期限切れ OAuth トークンを更新すると起動時に UI がフリーズする問題を修正
- フォークされた会話（`/fork`）が同じプランファイルを共有し、1 つのフォークでのプラン編集が他方を上書きする問題を修正
- 画像処理が失敗した際に Read ツールが過大な画像をコンテキストに入れ、長い画像中心セッションでそれ以降のターンが壊れる問題を修正
- heredoc コミットメッセージを含む複合 bash コマンドに対する誤判定の権限プロンプトを修正
- 複数の Claude Code インスタンスを実行した際にプラグインインストールが失われる問題を修正
- OAuth トークン更新後に claude.ai コネクタが再接続に失敗する問題を修正
- 以前に接続していたもののみではなく、org 設定済みコネクタすべてに対して claude.ai MCP コネクタ起動通知が表示される問題を修正
- 出力ファイルパスが欠落するバックグラウンドエージェント完了通知を修正。コンテキストコンパクション後に親エージェントがエージェントの結果を回復するのが困難だった
- コマンドが非ゼロステータスで終了した際の Bash ツールエラーメッセージの重複出力を修正
- ローカル Chrome のないマシンで動作した後、Chrome 拡張機能の自動検出が永続的に「not installed」でスタックする問題を修正
- マーケットプレイスがブランチ/タグ ref に pin されているときに `/plugin marketplace update` がマージコンフリクトで失敗する問題を修正
- `/plugin marketplace add owner/repo@ref` が `@` を誤解析する問題を修正。以前は `#` のみが ref セパレータとして機能し、`strictKnownMarketplaces` で診断不能なエラーを引き起こしていた
- 末尾スラッシュの有無で同じディレクトリが追加された際の `/permissions` Workspace タブの重複エントリを修正
- チームエージェントが設定されているとき `--print` が無期限にハングする問題を修正。exit ループが長寿命の `in_process_teammate` タスクを待たなくなった
- `ToolSearch` 呼び出しのたびに REPL に "❯ Tool loaded." が表示される問題を修正
- モデルが mingw スタイルのパスを使うときの Windows での `cd <cwd> && git ...` のプロンプトを修正
- ネイティブ画像プロセッサのロードを最初の使用まで遅延させて起動時間を改善
- ラップトップのスリープウェイク後のブリッジセッション再接続を、最大 10 分待つ代わりに数秒以内に完了するよう改善
- `/plugin uninstall` を改善。プロジェクトスコーププラグインを `.claude/settings.json` を変更する代わりに `.claude/settings.local.json` で無効化し、チームメイトに影響しないように
- プラグイン提供 MCP サーバーの重複排除を改善。手動設定されたサーバー（同じ command/URL）を複製するサーバーはスキップされ、重複接続とツールセットを防止。抑制は `/plugin` メニューに表示
- セッション中にデバッグログを切り替え可能にするよう `/debug` を更新。デバッグログはデフォルトで書き込まれなくなった
- 認証されていない org 登録の claude.ai コネクタの起動通知ノイズを削除

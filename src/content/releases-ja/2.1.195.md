- フルスクリーンモードでホイールスクロールを維持しつつマウスのクリック/ドラッグ/ホバーを無効化する `CLAUDE_CODE_DISABLE_MOUSE_CLICKS` を追加
- ハイフン付き識別子（例: `code-reviewer`、`mcp__brave-search`）を持つフックマッチャーが誤って部分一致していた問題を修正 — 完全一致になった。ハイフン付き MCP サーバーの全ツールにマッチするには `mcp__brave-search__.*` を使用する。
- macOS でデフォルト入力デバイスが変更された後、長時間のセッションで音声ディクテーションが無音を録音する問題を修正
- スペースなしで書かれる言語（日本語・中国語・タイ語）で音声ディクテーションの自動送信が一切発火しない問題を修正
- プロジェクトの `.claude/settings.json` のみで有効化された外部プラグインが、すべてのローダーパスで明示的なインストール同意を求めない問題を修正
- プラグインの `plugin.json` の `name` がマーケットプレイスのエントリ名と異なる場合に `/plugin` の有効化/無効化が機能しない問題を修正
- 新しい Claude Code バージョンで書き込まれたバックグラウンドジョブが `claude agents` から消えたりデータが失われたりする問題を修正
- クラッシュしたバックグラウンドタスクを再度開いたときに最大 5 秒間空白画面が表示される問題を修正
- コントロールソケットの起動失敗時にバックグラウンドエージェントデーモンが到達不能な状態で実行されて再起動がブロックされる問題を修正
- Linux での音声モードを改善: SoX がインストールされているがオーディオキャプチャデバイスが存在しない場合に「マイクなし」と「SoX未インストール」を区別するようになった
- `claude agents` の完了済みリストが利用可能な垂直スペースを埋めるように改善。短いターミナルではヘッダーがコンパクトになり、ライブセッションが常に表示される。
- プロビジョニングチェックリストを表示しながらコンテナが起動するよう Remote セッションのスタートアップを改善

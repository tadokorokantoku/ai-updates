- `claude mcp login <name>` と `claude mcp logout <name>` を追加。インタラクティブな `/mcp` メニューを開かずに CLI から MCP サーバーを認証可能。SSH 経由での認証完了をサポートする `--no-browser` stdin リダイレクトにも対応
- `/workflows` エージェント詳細ビューにステータスフィルタリングを追加（`f` キーで切り替え）
- `/plugin` の Installed タブに「Skills」セクションを追加
- `it2` CLI が見つからない場合に警告を出す `teammateMode: "iterm2"` 設定を追加
- `awsAuthRefresh` が設定されている場合、`/login` に「Claude Platform on AWS - refresh credentials」オプションを追加
- `!` bash コマンドが出力に対して Claude が自動応答するようになった。以前の動作（コンテキストへの追加のみ）に戻すには settings.json に `"respondToBashCommands": false` を設定
- マシンがスリープから復帰した後にストリーミングリクエストが「Content block not found」または JSON パースエラーで失敗する問題を修正
- サブエージェントのトランスクリプトのスクロール位置が終了時にメイントランスクリプトに影響する問題を修正
- バックグラウンドタスクのプレビューでエージェントのプランが読み込まれる前にツール名が生表示されるフラッシュを修正
- 並行 CLI セッションでプロダクト内パーミッションゲートが無効の場合に Chrome タブグループの分離が適用されない問題を修正
- バックグラウンドセッションのリキャップが重複する問題を修正。エージェント自身のターン終了時サマリーがリキャップ行として表示されるように
- `claude agents` からバックグラウンドセッションを開いたとき、直前の画面が背面に残る問題を修正
- 名前付きサブエージェントのスポーンに対して `Agent(type)` の拒否ルールと `Agent(x,y)` の許可タイプ制限が適用されない問題を修正
- メインターン終了後にバックグラウンドエージェントが実行中の場合、Esc と Ctrl+C が応答しない問題を修正
- パーミッションプロンプトでオプションテキストが折り返した場合にオプション番号がずれる問題を修正
- エージェントパネルで完了したサブエージェントの `x` キーによる非表示が機能しない問題を修正
- 古いセッションを再開したときに意図的に廃止されたツールに対して「MCP server disconnected」という誤解を招く通知が表示される問題を修正
- `/plugin` の Installed で先頭にスクロール済みの場合でも「more above」インジケータが表示される問題を修正
- アシスタントメッセージで `~~strikethrough~~` が打ち消し線としてレンダリングされずリテラルのチルダが表示される問題を修正
- コールド初回起動時にフラグが読み込まれる前に `--tools` でフィーチャーゲートされたツールが通過できてしまう問題を修正
- 返信後に `claude agents` のバックグラウンドジョブのステータスが古い「needs input」メッセージを表示し続ける問題を修正
- ライトターミナルで `claude agents` からバックグラウンドセッションを開く際にダークテーマが一瞬フラッシュする問題を修正
- `claude agents` でマウス選択したテキストを削除した後もハイライトが残る問題を修正
- 従量制の Enterprise・Team サブスクライバーでセッションコストが表示されない問題を修正
- エージェントチームの修正: tmux/pane バックエンド経由でスポーンされたチームメイトがリーダーの `--effort` レベルを継承するように
- Workflow の `agent({schema})` サブエージェントがスキーマバリデーションの繰り返し失敗で 5 回の試行後に中断せず無限ループする問題を修正
- `claude mcp get` と `claude mcp remove` を改善: タイポ時に最も近い設定済みサーバー名を提案し、長いサーバーリストを省略表示
- メモリの改善: サイズ制限に近づいたとき `MEMORY.md` インデックスを圧縮するようエージェントにリマインドするように
- スキルの frontmatter 改善: `display-name`、`default-enabled`、`fallback`、`metadata.*` キーが kebab-case、snake_case、camelCase を受け付けるように
- 不正な `SKILL.md` YAML frontmatter の処理を改善: サイレントに失敗する代わりに、空のメタデータでスキル本文を読み込むように
- `CLAUDE_CODE_MAX_RETRIES` の上限を 15 に変更。無人セッションには代わりに `CLAUDE_CODE_RETRY_WATCHDOG` を使用
- バックグラウンドサブエージェントが自動拒否する代わりにメインセッションでパーミッションプロンプトを表示するように変更。ダイアログにはどのエージェントが要求しているかが表示され、Esc でそのツールのみ拒否
- `/review <pr>` が `/code-review medium` と同じレビューエンジンを使用するように変更

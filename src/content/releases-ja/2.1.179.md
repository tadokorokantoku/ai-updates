- ストリーム途中の接続断を修正：部分的なレスポンスが生のエラーとして表示される代わりに保持されるようになり、"running tool" でスピナーが止まらなくなる問題も解消
- WSL2 の Windows Terminal および VS Code でのマウスホイールスクロールを修正（2.1.172 での退行）
- 大きなディレクトリツリーに対する sandbox の `denyRead`/`allowRead` glob が Bash ツールの説明を巨大にして Linux でセッションが使用不能になる問題を修正
- ターン完了直後にフィードバックサーベイが1桁の返答をセッション評価として即座に取得してしまう問題を修正
- ウェルカムスクリーンに複数のプロモーションバナーが重なって表示される問題を修正 — セッションあたり最大1つのプロモーションのみ表示されるように
- サブエージェントを表示している際に Ctrl+O でサブエージェントのトランスクリプトが表示されない問題を修正
- プロンプト入力欄をクリックしてもサブエージェント/フッターパネルからフォーカスが戻らない問題を修正
- リモートセッションのバックグラウンドタスクがターン間で "still running" として表示され続ける問題を修正
- リモートセッションでの plugin 読み込みパフォーマンスを改善

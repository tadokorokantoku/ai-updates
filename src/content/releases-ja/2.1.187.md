- サンドボックス化されたコマンドが認証情報ファイルやシークレット環境変数を読み取れないようにブロックする `sandbox.credentials` 設定を追加
- モデルピッカー、`--model`、`/model`、`ANTHROPIC_MODEL` に組織設定によるモデル制限を追加。制限されたモデルを選択した際に「restricted by your organization's settings」メッセージを表示
- フルスクリーンモードでの選択メニュー（パーミッションプロンプト、`/model`、`/config` など）にマウスクリックサポートを追加
- 元の `-p` 実行でモデルターンが生成されなかった場合に `--resume` が「No conversation found」で失敗する問題を修正
- `--json-schema` および Workflow の `agent({schema})` 構造化出力を修正: 成功した呼び出しの後にモデルが `StructuredOutput` を無制限に再呼び出しできなくなり、後続ターンで確実に構造化出力が返るように
- 5 分間応答のないリモート MCP ツール呼び出しが無制限にブロックする代わりにエラーで中断するように修正（`CLAUDE_CODE_MCP_TOOL_IDLE_TIMEOUT` で上書き可能）
- エージェントプロキシ CA のシステムトラストインストール追加後に Claude Code Remote セッションの起動が約 2.7 秒遅くなる問題を修正
- バイト単位の拡張キーイベントとしてペーストを送信するターミナルで貼り付けた韓国語・CJK テキストが文字化けする問題を修正
- Remote Control 経由で `/update` を実行する際に起動時の信頼ダイアログが表示されるべきタイミングでハングする問題を修正
- エージェントが構造化出力なしでターンを終了した場合に agents ビューのバックグラウンドジョブが「working」状態に無制限にスタックする問題を修正
- agents ビューへの移動・復帰後および `/bg`、`/tui`、`/update` 後にチャンネル接続が切断される問題を修正
- エージェント停止通知で停止者の帰属が正しく行われない問題を修正し、文言を改善（「came to rest」の代わりに「finished」/「stopped」）
- サブエージェントの深度追跡を修正: 再開されたサブエージェントが元のスポーン深度を復元し、フォークされたサブエージェントが深度上限にカウントされるように
- エージェントワークツリー登録のリーク修正: 強制終了されたエージェントの `.git/worktrees/` のロックされたエントリが自動的にクリーンアップされるように
- macOS の Ghostty でフルスクリーンモードで Cmd+click が URL を開かない問題を修正
- `claude --help` に `--bg`/`--background` フラグが表示されない問題を修正
- `/share` のアップロード中に Esc、Ctrl-C、Ctrl-D が機能しない問題を修正
- `/install-github-app` を改善: GitHub Actions ワークフローのセットアップがオプションになり、GitHub App のみをインストールしてワークフロー・シークレットの手順をスキップ可能に
- `/btw` を改善: ←/→ 矢印ナビゲーションで以前の回答をステップ移動できるように
- `/plugin` を改善: 最近使用していないプラグインを表示してクリーンアップしやすくなるように
- [VSCode] 大きなセッションを再開する際に拡張機能が応答しなくなる問題を修正

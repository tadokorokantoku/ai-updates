- API エラー（レート制限、認証失敗など）でターンが終了した際に発火する `StopFailure` フックイベントを追加
- プラグイン更新を生き残るプラグイン永続状態用の `${CLAUDE_PLUGIN_DATA}` 変数を追加。`/plugin uninstall` は削除前にプロンプト
- プラグイン同梱エージェント向けの `effort`、`maxTurns`、`disallowedTools` frontmatter サポートを追加
- ターミナル通知（iTerm2/Kitty/Ghostty ポップアップ、プログレスバー）が `set -g allow-passthrough on` 付きの tmux 内で実行中に外側のターミナルに到達
- 応答テキストが生成されるたびに行ごとにストリーミング
- Linux のサンドボックス化された Bash 内で `git log HEAD` が "ambiguous argument" で失敗し、スタブファイルが作業ディレクトリの `git status` を汚染する問題を修正
- サブエージェントを使用する大きなセッション（>5 MB）で `cc log` と `--resume` が会話履歴を無言で切り詰める問題を修正
- API エラーが stop フックをトリガーし、ブロックエラーをモデルに再供給する際の無限ループを修正
- `deny: ["mcp__servername"]` 権限ルールが MCP サーバーツールをモデルに送信する前に削除せず、ブロックされたツールを見て試行できる問題を修正
- `sandbox.filesystem.allowWrite` が絶対パスで動作しない問題を修正（以前は `//` プレフィックスが必要）
- macOS 版の `/sandbox` Dependencies タブが macOS 固有情報ではなく Linux 前提条件を表示する問題を修正
- **セキュリティ:** `sandbox.enabled: true` が設定されているが依存関係が欠けている際の無言のサンドボックス無効化を修正 — 可視の起動警告を表示
- `bypassPermissions` モードで `.git`、`.claude`、その他の保護ディレクトリがプロンプトなしで書き込み可能な問題を修正
- 通常モードでの ctrl+u が readline の kill-line の代わりにスクロールする問題を修正（ctrl+u/ctrl+d ハーフページスクロールはトランスクリプトモード専用に）
- voice モードの modifier 組み合わせ push-to-talk キーバインド（例: ctrl+k）が即座にアクティベートせず hold が必要な問題を修正
- WSLg (Windows 11) 上の WSL2 で voice モードが動作しない問題を修正。WSL1/Win10 ユーザーには明確なエラーを表示
- `--worktree` フラグが worktree ディレクトリからスキルとフックをロードしない問題を修正
- `CLAUDE_CODE_DISABLE_GIT_INSTRUCTIONS` と `includeGitInstructions` 設定がシステムプロンプトの git status セクションを抑制しない問題を修正
- VS Code が Dock/Spotlight から起動されたときに Bash ツールが Homebrew やその他の PATH 依存バイナリを見つけられない問題を修正
- truecolor サポートをアドバタイズしない VS Code/Cursor/code-server ターミナルで Claude のオレンジ色が washed-out になる問題を修正
- `/model` ピッカーにカスタムエントリを追加する `ANTHROPIC_CUSTOM_MODEL_OPTION` 環境変数を追加。表示用に `_NAME` と `_DESCRIPTION` の接尾辞付き変数もオプションで指定可能
- Haiku モデルを使用時に `ANTHROPIC_BETAS` 環境変数が無言で無視される問題を修正
- キューされたプロンプトが改行セパレータなしで連結される問題を修正
- 大きなセッションを再開する際のメモリ使用量と起動時間を改善
- [VSCode] 既に認証済みのときにサイドバーを開くと一瞬ログイン画面がちらつく問題を修正
- [VSCode] Opus を選択した際の「API Error: Rate limit reached」を修正 — プランティアが不明なサブスクライバーにモデルドロップダウンが 1M コンテキストバリアントを提供しなくなった

- Claude が有用なコンテキストを自動メモリに自動保存。/memory で管理
- コードブロックが存在する場合にインタラクティブピッカーを表示する `/copy` コマンドを追加。個別のコードブロックまたは完全な応答を選択可能
- 複合 bash コマンド（例: `cd /tmp && git fetch && git push`）の「always allow」プレフィックス候補を改善。コマンド全体を 1 つとして扱うのではなく、より賢いサブコマンドごとのプレフィックスを計算
- 短いタスクリストの順序を改善
- 完了したサブエージェントタスク状態を解放することでマルチエージェントセッションのメモリ使用量を改善
- 複数の Claude Code インスタンスを同時に実行した際の MCP OAuth トークン更新のレース条件を修正
- 作業ディレクトリが削除された際、シェルコマンドが明確なエラーメッセージを表示しない問題を修正
- 複数の Claude Code インスタンスが同時に実行された際に認証を消してしまう可能性のある設定ファイル破損を修正

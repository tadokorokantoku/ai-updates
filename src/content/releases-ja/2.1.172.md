- サブエージェントが独自のサブエージェントを生成できるようになった（最大5階層まで）
- Amazon Bedrock が `AWS_REGION` 未設定時に `~/.aws` 設定ファイルから AWS リージョンを読み込むようになった。AWS SDK の優先順位に準拠し、`/status` でリージョンの取得元を確認できる
- `/plugin` でマーケットプレイスのプラグイン一覧を閲覧する際に検索バーを追加
- `claude_code.lines_of_code.count` OTEL メトリクスに `model` 属性を追加
- usage credits なしで 1M コンテキストを使用しているセッションが永久にスタックする問題を修正。セッションが自動的に標準コンテキスト上限以下にコンパクトされるようになった
- 複数の画像を含む会話で「会話内の画像を処理できなかったため削除されました」エラーが繰り返し表示される問題を修正
- ワーカーが応答した後もエージェントビューがセッションを「Working」のスピナー付きで最大30秒間表示し続ける問題を修正
- バックグラウンドエージェントが事前ウォーム済みワーカーにディスパッチされた際に、別ディレクトリのプロジェクト設定（`.mcp.json` 承認、trust）を読み込む可能性がある問題を修正
- デーモンの自動更新後、旧バージョンで開始したセッションへのバックグラウンドセッション接続が EAUTH で失敗する問題を修正
- 入れ子エージェントが停止した後、バックグラウンドのサブエージェントがエージェントパネルで「active」のままスタックする問題を修正
- `claude agents` のディスパッチ入力で `/model` サジェストがスラッシュプレフィックス付きで表示され、組織で無効化されたモデルも表示されていた問題を修正
- `availableModels` の制限がサブエージェントのモデルオーバーライド、エージェントディスパッチモデルピッカー、アドバイザーモデルに適用されない問題を修正
- `availableModels` の許可リストに `claude-opus-4-8` などのバージョン固有 ID が含まれている場合、`/model` ピッカーの Opus・Sonnet 1M 行が非表示になる問題を修正
- Bedrock の `/model` ピッカーがプロバイダーが提供していないモデルを表示していた問題を修正。選択すると黙ってセッションモデルが切り替わり、複数行に選択マーカーが点灯していた
- `ANTHROPIC_DEFAULT_OPUS_MODEL` に 1M コンテキストサフィックスが既に含まれている場合、モデル ID に `[1M][1m]` が重複付与される問題を修正
- `opusplan` モデル設定がプラン対象ユーザーのプランモードで 1M コンテキストで起動しない問題を修正。`opusplan[1m]` のワークアラウンドもプランモードで Opus に正しく切り替わるようになった
- `WebFetch(domain:*.example.com)` のワイルドカードドメインルールが allow・deny・ask のいずれの位置でもサブドメインにマッチしない問題、およびパターン途中のワイルドカード（例: `Read(secrets-*/config.json)`）を含むファイルパーミッションルールが起動時に拒否される問題を修正
- サブエージェントのチャットタブが開いているときに上矢印でプロンプト履歴を表示するとメインエージェントのプロンプトが表示される問題を修正
- リモートセッションでマウントされたチームメモリストア（`CLAUDE_MEMORY_STORES`）がメモリ呼び出しで見つからない問題を修正
- プロンプト文字列やコメントで `Date.now()`・`Math.random()` に言及しているだけのスクリプトをワークフロー検証が拒否する問題を修正
- 完全対応していない Windows コンソールでのマウストラッキングを無効化
- `/plugin` マーケットプレイスリストで長いプラグインリストから戻った後にカーソルが消える問題、および Esc でプラグインブラウザを抜けると誤ったタブに戻る問題を修正
- 冗長なメッセージ正規化を削除し、ストリーミングツール使用状態が変化しない場合のメッセージ履歴フル変換を回避することで、長い会話でのパフォーマンスを改善
- アイドル時の CPU 使用率を低減：`/goal` ステータスチップのアイドル時 5Hz ターミナル再描画を停止し、サブエージェント並列実行中の UI 再描画も削減
- Claude in Chrome のツール読み込みを改善：ブラウザツールをツールごとの個別呼び出しではなく単一のバッチ呼び出しで読み込むように変更
- 非インタラクティブモードでの利用規約拒否メッセージを改善し、新しいセッションの開始またはモデルの変更を提案するようにした
- `/code-review` で claude.ai にサインインしていない場合でも `ultra` オプションが表示されるようになった。クラウドレビューには claude.ai アカウントが必要である旨の説明が表示される
- Remote Control フッターインジケーターを "/rc active" に短縮し、狭いターミナルでは非表示に
- リモートセッションでの `/loop` の推奨を停止。保留中のループはコンテナをアクティブに保持しない
- [VSCode] PowerShell のツール呼び出しが適切なコマンド表示と権限ダイアログではなく生の JSON としてレンダリングされる問題を修正し、表示されるシェル出力から ANSI エスケープコードを除去

- ツール呼び出しの失敗と拒否をデバッグログに追加
- gateway ユーザー向けの context management 検証エラーを修正。`CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1` でエラー回避可能に
- 特定の GitHub PR 番号または URL に紐づくセッションを再開する `--from-pr` フラグを追加
- `gh pr create` 経由で作成されたセッションを自動的に PR にリンク
- /context コマンドが色付き出力を表示しない不具合を修正
- PR ステータス表示時にステータスバーがバックグラウンドタスクインジケータを重複表示する不具合を修正
- Windows: `.bashrc` ファイルを持つユーザーで bash コマンド実行が失敗する不具合を修正
- Windows: 子プロセス生成時にコンソールウィンドウが点滅する不具合を修正
- VSCode: 長時間セッション後に OAuth トークンの有効期限切れで 401 エラーが発生する不具合を修正

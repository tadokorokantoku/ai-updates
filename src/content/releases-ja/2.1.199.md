- API がストリーミング中に部分出力後に overloaded/server エラーを返した際、ストリーミング応答が破棄されていた問題を修正 — 部分的な応答は不完全であることを示す通知とともに保持されるように
- レートリミットやサーバーエラーで中断されたサブエージェントが、部分的な作業結果を親に返さず無言で失敗していた問題を修正

- macOS で SSH 経由のバックグラウンドエージェントが「Could not switch to audit session」でコールドスタートに失敗する問題を修正 (2.1.196 でのリグレッション)

- バックグラウンドジョブが長時間のコマンドを実行している間、進捗インジケータが数分間止まってしまう問題を修正
- メモリ不足のマシンでバックグラウンドセッションが汎用エラーを表示していた問題を修正 — メモリ不足であることを示し、リソースの解放を提案するように
- バックグラウンドエージェントが完了した際、リモートセッションが agent view で Working と Idle の間を一瞬フリップフリップする問題を修正
- 他のサブエージェントが作業中の間、アイドル状態のサブエージェントが agent panel から消えてしまう問題を修正。余剰のアイドルエージェントは展開可能なサマリー行にまとめられるように
- サブエージェントを表示中に `/model` や `/fast` を入力すると、無言でリードのモデルピッカーが開いてしまう問題を修正 — コマンドがリードに適用されることを説明する通知が表示されるように
- `SessionStart`、`Setup`、`SubagentStart` フックが終了コード2で終了した際に stderr が無言で隠されていた問題を修正 — エラーがトランスクリプトに表示されるように
- `claude --dangerously-skip-permissions daemon <subcommand>` がサブコマンドを実行せずチャットプロンプトとして扱われていた問題を修正
- 再起動したエージェントが以前のエージェントの名前を再利用した際に `SendMessage` が無言で誤ったルーティングをしていた問題を修正 — 不一致を検出し、呼び出し元に再指定を求めるように

- 起動時リカバリダイアログから破損した設定ファイルをリセットすると、復元不能な形で破壊されていた問題を修正 — 事前にファイルをバックアップするように

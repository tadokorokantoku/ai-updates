## 変更内容

- `claude agents` に新フラグを追加：`--add-dir`、`--settings`、`--mcp-config`、`--plugin-dir`、`--permission-mode`、`--model`、`--effort`、`--dangerously-skip-permissions` — ディスパッチされたバックグラウンドセッションの設定に使用
- Fast モードのデフォルトが Opus 4.7 に変更（以前は Opus 4.6）。`CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1` を設定すると Opus 4.6 に固定できる
- ルートに `SKILL.md` があり `skills/` サブディレクトリがない plugin が skill として認識されるようになった
- `/plugin` 詳細ペインと `claude plugin details` に、plugin が提供する LSP サーバーが表示されるようになった
- `/web-setup` が既存の GitHub App 接続を置き換える前に警告を表示するようになった
- `MCP_TOOL_TIMEOUT` がリモート HTTP・SSE MCP サーバーのリクエストごとのフェッチタイムアウトに反映されない問題を修正 — 設定値に関わらずツール呼び出しが 60 秒でキャップされていた
- バックグラウンドセッションが既存の git worktree を認識できず、Edit がブロックされ EnterWorktree が重複作成を拒否していた問題を修正
- macOS のスリープ/ウェイク後にバックグラウンドセッションが消え、daemon の再接続が失敗する問題を修正 — daemon がクロックのジャンプを経過アイドル時間として扱わず検出するようになった
- バイナリのアップグレード時（例：`brew upgrade`）に daemon が正常終了せず、ディスパッチされたエージェントが削除済みパスでクラッシュループする問題を修正
- Claude-in-Chrome extension が共有タブなしで接続している場合にバックグラウンドエージェントがクラッシュループする問題を修正
- アタッチ中の `claude agents` セッションでリンクをクリックする際、バックグラウンドワーカーのヘッドレスブラウザシムが適用される問題を修正
- `claude agents` の「v でエディタを開く」がシェルの `$EDITOR`/`$VISUAL` ではなく daemon のデフォルトエディタを使用していた問題を修正
- `claude agents` がネットワークドライブの作業ディレクトリで Windows にてデッドロックする問題を修正；起動中に Ctrl+C が機能するようになった
- Apple Terminal や 256 色のみ対応ターミナルから `claude agents` セッションにアタッチした際の背景色にじみを修正
- `claude --bg --dangerously-skip-permissions` がリタイア/ウェイク後に維持されない問題を修正
- 最初のメッセージがリンクの場合にセッションタイトルが URL から生成される問題を修正
- リモートクライアントからの冗長な `set_model` リクエストがトランスクリプトに `/model` のパンくずを重複挿入する問題を修正
- `skills: ["./"]` を使う plugin が「パスが plugin ディレクトリ外に出る」という誤ったエラーを表示する問題を修正
- インストールメタデータが存在しない場合に plugin キャッシュのクリーンアップがアクティブな plugin バージョンディレクトリを削除する問題を修正
- `/plugin` ブラウズペインで新規公開 plugin が「0 installs」と表示される問題を修正
- plugin アドバイザリがデフォルトフォルダをシャドウする `plugin.json` のキーをすべて列挙しない問題を修正
- リアクティブコンパクションの改善：最初の要約試行が元のリクエストのオーバーフローサイズを初期値として使用するようになり、ほぼフルコンテキストでの無駄な再試行を回避
- hook 設定エラーの改善：`SessionStart`/`Setup`/`SubagentStart` に対して prompt または agent タイプの hook を設定した場合に「コマンドタイプの hook を使用してください」という明確なエラーを表示するようになった
- Usage Policy 拒否メッセージから古い `/model claude-sonnet-4-20250514` の提案を削除

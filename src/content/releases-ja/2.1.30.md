  - "現在のセッションのトラブルシューティングを支援する `/debug` コマンドを追加"

- Read ツールの PDF 向けに `pages` パラメータを追加。特定ページ範囲を読み取れるように (例: `pages: "1-5"`)。10 ページ超の大きな PDF は `@` で参照した際にコンテキストに展開せず軽量な参照を返すように
- Dynamic Client Registration をサポートしない MCP サーバー (例: Slack) 向けに事前構成された OAuth クライアント資格情報を追加。`claude mcp add` で `--client-id` と `--client-secret` を使用
- 現在のセッションのトラブルシューティングを Claude が支援する `/debug` を追加
- 読み取り専用モードでの `git log` および `git show` の追加フラグ (例: `--topo-order`, `--cherry-pick`, `--format`, `--raw`) をサポート
- Task ツールの結果にトークン数、ツール使用回数、継続時間のメトリクスを追加
- リデュースモーション (動きを減らす) モードを config に追加
- API 会話履歴に表示される幻のような "(no content)" テキストブロックを修正。トークンの無駄と潜在的なモデル混乱を軽減
- ツール名が変更された場合のみ prompt キャッシュが無効化され、ツールの説明や入力スキーマ変更時に無効化されない不具合を修正
- 会話に thinking ブロックが含まれるときに `/login` 後に 400 エラーが発生する不具合を修正
- `parentUuid` サイクルを含む破損したトランスクリプトファイルからセッションを再開した際のハングを修正
- extra-usage が利用できない Max 20x ユーザーに対してレート制限メッセージが誤って "/upgrade" を提案する不具合を修正
- 入力中に権限ダイアログがフォーカスを奪う不具合を修正
- SDK 提供の MCP ツールが共有アプリケーション状態に同期されず、サブエージェントがアクセスできない不具合を修正
- `.bashrc` ファイルを持つ Windows ユーザーが bash コマンドを実行できなくなっていたリグレッションを修正
- `--resume` のメモリ使用量を改善 (多数セッションを持つユーザーで 68% 削減)。セッションインデックスを stat ベースの軽量な読み込みと段階的なエンリッチメントに置き換え
- `TaskStop` ツールを改善。汎用的な "Task stopped" メッセージではなく停止したコマンド/タスクの説明を結果行に表示
- `/model` をキューに入れずに即時実行するよう変更
- [VSCode] 質問ダイアログの "Other" テキスト入力で複数行入力をサポート (改行は Shift+Enter)
- [VSCode] 新しい会話開始時にセッションリストに重複セッションが現れる不具合を修正

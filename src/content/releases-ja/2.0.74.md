- 定義へジャンプ、参照の検索、ホバードキュメントなどのコードインテリジェンス機能用に LSP (Language Server Protocol) ツールを追加
- Kitty、Alacritty、Zed、Warp ターミナル向けの `/terminal-setup` サポートを追加
- `/theme` でシンタックスハイライトのオン/オフを切り替える ctrl+t ショートカットを追加
- テーマピッカーにシンタックスハイライト情報を追加
- macOS ユーザーがターミナル設定により Alt ショートカットが動作しないときのガイダンスを追加
- スキルの `allowed-tools` がスキルから呼び出されるツールに適用されない問題を修正
- ユーザーが既に Opus を使用しているときに Opus 4.5 のヒントが誤って表示される問題を修正
- シンタックスハイライトが正しく初期化されていないときに発生する可能性のあるクラッシュを修正
- `/plugins discover` でリスト選択インジケーターが検索ボックスにフォーカスがあるときに表示される視覚的バグを修正
- macOS のキーボードショートカットで 'alt' の代わりに 'opt' を表示するように修正
- `/context` コマンドの可視化を改善。スキルとエージェントをソース別にグループ化、スラッシュコマンドを整理、トークン数でソート
- [Windows] レンダリング不良の問題を修正
- [VSCode] 年末プロモーションメッセージ用にギフトタグのピクトグラムを追加

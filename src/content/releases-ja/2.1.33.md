- tmux での agent teammate セッションのメッセージ送受信を修正
- agent teams が現在のプランで利用できないという警告を修正
- マルチエージェントワークフロー向けに `TeammateIdle` と `TaskCompleted` の hook イベントを追加
- エージェントの "tools" frontmatter で `Task(agent_type)` 構文によりどのサブエージェントを生成可能にするか制限する機能を追加
- エージェントに `memory` frontmatter フィールドのサポートを追加。`user`, `project`, `local` スコープで永続メモリを有効化
- 発見しやすさ向上のため、スキル説明および `/skills` メニューにプラグイン名を追加
- 拡張思考中に新しいメッセージを送信すると思考フェーズが中断される不具合を修正
- ストリーム途中で中断した際に空白テキストと thinking ブロックが組み合わさって正規化を回避し無効なリクエストを生成する API エラーを修正
- ストリーミングエンドポイントでの 404 エラーが非ストリーミングフォールバックをトリガーしないようにした API プロキシ互換性問題を修正
- `settings.json` の環境変数で構成されたプロキシ設定が Node.js ビルド上の WebFetch やその他 HTTP リクエストに適用されない不具合を修正
- `/resume` セッションピッカーがスラッシュコマンドで開始されたセッションに対してクリーンなタイトルの代わりに生の XML マークアップを表示する不具合を修正
- API 接続失敗時のエラーメッセージを改善。汎用的な "Connection error" ではなく具体的な原因 (例: ECONNREFUSED, SSL エラー) を表示
- 無効な managed settings によるエラーを表面化
- VSCode: リモートセッションのサポートを追加。OAuth ユーザーが claude.ai からセッションをブラウズ・再開可能に
- VSCode: セッションピッカーに git ブランチとメッセージ数を追加。ブランチ名での検索をサポート
- VSCode: 初回セッション読み込みおよびセッション切り替え時の scroll-to-bottom のアンダースクロールを修正

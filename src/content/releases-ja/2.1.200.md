- `AskUserQuestion` ダイアログがデフォルトで自動継続しないように変更。アイドルタイムアウトを使いたい場合は `/config` から opt-in
- CLI、`--help`、VS Code、JetBrains 全体で "default" パーミッションモードの名称を "Manual" に変更。`--permission-mode manual` と `"defaultMode": "manual"` が `default` と併用可能に
- `.claude.json` の `disabledMcpServers` または `enabledMcpServers` が配列以外の値に設定されていると起動時にクラッシュしていた問題を修正
- スリープ/ウェイク後やスタックしたセッションの再開時に、バックグラウンドセッションがターンの途中で無言で停止してしまう問題を修正
- スタック後の respawn で、Esc でキャンセルしたターンがバックグラウンドセッションで再実行されてしまう問題を修正
- クラッシュにより残った古い `daemon.lock` の PID を OS が再利用した場合、バックグラウンドエージェントが二度と起動しなくなる問題を修正
- バックグラウンドエージェントデーモンの引き継ぎ処理を修正。再インストールされた古いビルドがデーモンを乗っ取れないように — ビルドの新しさはバージョンに埋め込まれたビルドタイムスタンプで判定するように
- バックグラウンドエージェントの名簿(roster)に関する問題を修正: 一時的な破損が孤児クリーンアップを恒久的に無効化していた問題、古いバイナリが新しいバージョンで書き込まれたフィールドを保持できていなかった問題、デーモン再起動時にソケット認証トークンが失われていた問題
- テキスト出力を1つも生成する前にレートリミットで中断されたサブエージェントが、正しく失敗する代わりに空の結果を返していた問題を修正
- バックグラウンドエージェントの出力に含まれる制御バイトが agent view のターミナルに届いてしまう問題を修正
- `claude agents --plugin-dir <dir>` で、フラグを `agents` の後に置いた場合にプラグインの agents と skills が agent view に表示されない問題を修正
- 同一リポジトリの git worktree からプロジェクトスコープのプラグインが正しく読み込まれない問題を修正
- `/mcp` のサーバー一覧がスクリーンリーダーや拡大鏡でフォーカスを追跡していなかった問題を修正
- 録音に音声が含まれない場合に、音声入力が誤解を招く「Voice connection failed」を表示していた問題を修正
- 同期されたターミナル出力を有効にすることで tmux 3.4+ でのレンダリングのちらつきを修正
- スクリーンリーダー出力を改善: 装飾的なグリフを非表示にし、トランスクリプトの記号を短いラベルとして読み上げ、ネストしたテーブルを `Header: value.` の行として読み上げるように
- システムのメモリ不足でインストールが強制終了された場合に、その旨を説明するようインストールスクリプトを改善

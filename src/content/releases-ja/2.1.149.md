- `/usage` がスキル・サブエージェント・プラグイン・MCP サーバーごとのカテゴリ別内訳を表示するようになった
- `/diff` の詳細ビューをキーボードでスクロールできるようになった（矢印キー、`j`/`k`、`PgUp`/`PgDn`、`Space`、`Home`/`End`）
- Markdown 出力で GFM タスクリストのチェックボックス（`- [ ] todo` / `- [x] done`）が箇条書きではなく正しくレンダリングされるようになった
- Enterprise: `managed-mcp.json` と並行して claude.ai クラウド MCP コネクタを読み込む `allowAllClaudeAiMcps` マネージド設定を追加した
- PowerShell の権限バイパスを修正した: 組み込みの `cd` 関数（`cd..`、`cd\`、`cd~`、`X:`）が検出されずに作業ディレクトリを変更し、後続のコマンドがワークスペース外を読み取れた問題を修正した
- git worktree における sandbox の書き込み許可リストが、共有 `.git` ディレクトリのみではなくメインリポジトリのルート全体をカバーしていた問題を修正した（`hooks/` と `config` は拒否対象）
- PowerShell のプレフィックス/ワイルドカード許可ルール（例: `PowerShell(dotnet.exe build *)`）がネイティブ実行ファイルとスクリプトを事前承認しない問題を修正した
- `cd`/`pushd`/`popd` をまたいで `PWD`/`OLDPWD`/`DIRSTACK` の古い変数トラッキング値を信頼していた権限解析の問題を修正した
- Bash ツールの `find` コマンドが大きなディレクトリツリーで macOS のシステムファイル/vnode テーブルを枯渇させてホストをクラッシュさせる問題を修正した
- マネージド設定の承認ダイアログが起動時に承認後もターミナルをフリーズさせる問題を修正した
- 作業ツリーに実際の変更がない場合に `/ultraplan` とリモートセッション作成が「Could not capture uncommitted changes」で失敗する問題を修正した
- スクリプトパスにスペースが含まれる場合に `otelHeadersHelper` が無音で失敗する問題を修正した（ヘルパーの失敗は `/doctor` とデバッグログに報告されるようになった）
- 思考スピナーがツール呼び出しをまたいで新たな思考バーストにアンバー色のまま残る問題を修正した
- 短い行が多い出力で折りたたまれた Bash 出力が非表示行数を誤って報告する問題を修正した
- ヒントが入力ボックスをオーバーフローするときにスラッシュコマンドの引数ヒントが末尾の入力文字をクリップする問題を修正した
- ディレクトリベース名と異なる `name:` frontmatter を持つスキルをタブ補完した後に引数ヒントと漸進的な引数候補が表示されない問題を修正した
- ステータスバーがスキル/エージェントの `effort:` frontmatter で適用されたエフォートレベルではなく、ユーザーの基準 `/effort` 設定を表示する問題を修正した
- Ctrl+O のトランスクリプトビューが開いた時点でフリーズして新しいメッセージを追跡しない問題を修正した
- 呼び出したプロンプト履歴エントリを編集した後、矢印キーでさらに上下に移動すると編集内容が失われる問題を修正した
- 関係のない設定を切り替えたときに `/config` の終了サマリーが auto-compact とテーマへの幽霊変更を報告する問題を修正した
- キャッシュされたセッションメタファイルにオプションフィールドが欠落している場合に `/insights` がクラッシュする問題を修正した
- 入力が欠落した不正な PowerShell と History ツール呼び出しがトランスクリプト折りたたみで読み取りと誤分類される問題を修正した
- claude.ai または Claude モバイルアプリから Remote Control セッションを名前変更しても `claude --resume` 用のローカルセッション名が更新されない問題を修正した
- 送信直後のプロンプトが上矢印履歴に二重表示される競合状態を修正した
- フルスクリーンモードで「Jump to bottom」ピルをタップしても即座に消えない問題を修正した
- `/feedback` レポートにコンテキストコンパクション前の会話を含めるよう改善した（長いセッションの初期に発生した問題のトリアージが容易になった）

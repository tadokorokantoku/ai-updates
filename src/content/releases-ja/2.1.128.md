- `EnterWorktree` がドキュメント通り `origin/<default-branch>` ではなくローカル HEAD から新しいブランチを作成するように修正 — 未プッシュのコミットが失われなくなった
- Auto モード：クラシファイアがアクションを評価できない場合、エラーメッセージにヒント（リトライ、`/compact`、または `--debug` での実行）が含まれるように
- フォーカスモードで新しいプロンプトを送信した際に前のレスポンスが一瞬暗くなる問題を修正
- OSC 9 を通知として解釈する Kitty など一部ターミナルで `/exit` のたびに "4;0;" デスクトップ通知が出る問題を修正
- レート制限時に Remote Control が実行可能なアップセルオプションではなく空の "Opening your options…" メッセージを表示していた問題を修正
- 画像の読み込みに失敗した際にドラッグ＆ドロップ画像アップロードが "Pasting text…" でハングする問題を修正
- 非常に大きな入力（>10 MB）を `claude -p` に stdin で渡した際のクラッシュループを修正
- フルスクリーンモードで折り返した各行の長い URL が個別にクリックできなかった問題を修正
- `--plugin-dir` でロードしたプラグインの `/plugin` Components パネルに "Marketplace 'inline' not found" と表示される問題を修正
- MCP ツールの結果で、サーバーが structured content と content blocks の両方を返した場合に画像が失われる問題を修正
- リスト項目内のフェンスコードブロックをコピーペーストした際に先頭の空白が混入する問題を修正
- `/config` のタブナビゲーションでフォーカスが失われる問題を修正 — タブヘッダーにフォーカスが残り、矢印キーと Esc が引き続き機能するように
- OSC 8 ハイパーリンクをサポートしないターミナルで markdown リンクのラベルが失われる問題を修正 — リンクが URL のみではなく `label (url)` 形式で表示されるように

- 並列シェルツールコール時に読み取り専用コマンド（grep、git diff、ls）の失敗が兄弟コールをキャンセルしていた問題を修正
- effort をサポートしないモデルでバナーに "with X effort" と表示される問題を修正
- サードパーティプロバイダーで `/fast` が "not available" ではなく無関係なスキルにファジーマッチしていた問題を修正
- Bedrock のデフォルトモデルがリージョン適切なプレフィックスではなく `global.*` に解決される問題を修正
- vim モード：NORMAL モードで `Space` がカーソルを右に移動するように修正（標準的な vi/vim の挙動に合わせた）
- ターミナルの進捗インジケーター（OSC 9;4）がツールコール間で消灯する問題を修正 — ターン全体を通じて表示されるように

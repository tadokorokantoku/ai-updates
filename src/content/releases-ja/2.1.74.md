- `/context` コマンドに実行可能な提案を追加 — コンテキストが重いツール、メモリの肥大、容量警告を特定し、具体的な最適化のヒントを提示
- 自動メモリの保存用にカスタムディレクトリを設定する `autoMemoryDirectory` 設定を追加
- ジェネレータが早期に終了してもストリーミング API レスポンスバッファが解放されず、Node.js/npm コードパスで無制限の RSS 成長を引き起こすメモリリークを修正
- 管理ポリシーの `ask` ルールがユーザーの `allow` ルールやスキルの `allowed-tools` でバイパスされる問題を修正
- エージェント frontmatter の `model:` フィールドと `--agents` JSON 設定で完全なモデル ID（例: `claude-opus-4-5`）が無言で無視される問題を修正。エージェントが `--model` と同じモデル値を受け入れるように
- コールバックポートがすでに使用中のときに MCP OAuth 認証がハングする問題を修正
- refresh token の期限切れ後に MCP OAuth が再認証を求めない問題を修正。HTTP 200 でエラーを返す OAuth サーバー（例: Slack）向け
- ターミナルに一度もマイク権限が付与されたことのないユーザー向けに、macOS ネイティブバイナリで voice モードが無言で失敗する問題を修正。バイナリに `audio-input` entitlement を含めて macOS が正しくプロンプトを表示するように
- `hook.timeout` に関係なく終了時に 1.5 秒後に `SessionEnd` フックが終了される問題を修正。`CLAUDE_CODE_SESSIONEND_HOOKS_TIMEOUT_MS` で設定可能に
- ローカルソースを持つマーケットプレイスプラグインに対して REPL 内で `/plugin install` が失敗する問題を修正
- マーケットプレイス更新が git サブモジュールを同期しない問題を修正。サブモジュール内のプラグインソースは更新後に壊れなくなった
- 引数付きの未知のスラッシュコマンドが入力を無言で落とす問題を修正。入力を警告として表示するように
- Windows Terminal、conhost、VS Code 統合ターミナルでヘブライ語、アラビア語、その他の RTL テキストが正しくレンダリングされない問題を修正
- 不正な file URI により Windows で LSP サーバーが動作しない問題を修正
- `--plugin-dir` を変更。ローカル開発コピーが同じ名前のインストールされたマーケットプレイスプラグインを上書きするように（管理対象設定によってそのプラグインが強制有効化されていない限り）
- [VSCode] Untitled セッションで削除ボタンが動作しない問題を修正
- [VSCode] ターミナル対応の加速度で統合ターミナルのスクロールホイール応答性を改善

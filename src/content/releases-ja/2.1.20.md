- vim ノーマルモードでカーソルがそれ以上移動できないときに矢印キーで履歴ナビゲーションを追加
- 外部エディタのショートカット (Ctrl+G) をヘルプメニューに追加し発見しやすさを向上
- PR レビュー状態インジケータをプロンプトフッタに追加。現在のブランチの PR 状態 (approved, changes requested, pending, draft) を色付きドットとクリック可能なリンクで表示
- `--add-dir` フラグで指定した追加ディレクトリからの `CLAUDE.md` ファイルの読み込みをサポート (`CLAUDE_CODE_ADDITIONAL_DIRECTORIES_CLAUDE_MD=1` の設定が必要)
- `TaskUpdate` ツールからタスクを削除する機能を追加
- セッションコンパクト化の不具合を修正。resume 時にコンパクトサマリではなく完全な履歴を読み込む場合があった
- タスク作業中に送信されたユーザーメッセージをエージェントが無視することがあった不具合を修正
- ワイド文字 (絵文字、CJK) のレンダリングアーティファクトを修正。狭い文字で置換されたとき末尾の列がクリアされない問題
- MCP ツールのレスポンスに特殊な Unicode 文字が含まれるときの JSON パースエラーを修正
- 複数行および折り返したテキスト入力で上下矢印キーが履歴ナビゲーションよりもカーソル移動を優先するように修正
- UP 矢印でコマンド履歴をナビゲートする際にドラフトプロンプトが失われる不具合を修正
- スラッシュコマンドを入力途中で打つとゴーストテキストがちらつく不具合を修正
- marketplace ソース削除時に設定が正しく削除されない不具合を修正
- `/context` など一部コマンドでの出力重複を修正
- タスクリストがメイン会話ビューの外に表示されることがある不具合を修正
- Python docstring のような複数行構造内で発生する diff のシンタックスハイライトを修正
- ツール使用キャンセル時のクラッシュを修正
- `/sandbox` コマンド UI を改善。依存関係が欠けている場合にインストール手順とともに依存状態を表示
- 思考中のステータステキストを控えめなシマーアニメーション付きに改善
- 端末の高さに応じて表示項目を動的に調整するようタスクリストを改善
- フォーク会話のヒントを改善し、元のセッションを再開する方法を表示
- 折りたたまれた read/search グループを進行中は現在形 ("Reading", "Searching for")、完了後は過去形 ("Read", "Searched for") で表示するよう変更
- `ToolSearch` の結果を会話内インラインではなく簡潔な通知として表示するよう変更
- MCP ツールで設定されている場合に `/commit-push-pr` スキルが PR URL を自動的に Slack チャンネルへ投稿するよう変更
- `/copy` コマンドを全ユーザーで利用可能に変更
- バックグラウンドエージェントが起動前にツール権限をプロンプトするよう変更
- `Bash(*)` のような権限ルールを受け付け、`Bash` と同等に扱うよう変更
- 設定バックアップをタイムスタンプ付きでローテート (最新 5 件保持) しデータ損失を防ぐよう変更

- セッションタイトルが会話の言語で生成されるようになりました（`language` 設定で言語を固定できます）
- フッター行に regex でマッチするリンクバッジを表示する `footerLinksRegexes` 設定を追加（ユーザーまたはマネージド設定から構成可能）
- Bedrock 認証情報キャッシュを改善：`awsCredentialExport` からの認証情報が固定の1時間ではなく `Expiration` までキャッシュされるようになりました
- `availableModels` の適用を修正：エイリアスモデルの選択が `ANTHROPIC_DEFAULT_*_MODEL` 環境変数経由でブロック対象モデルにリダイレクトされなくなり、`/fast` もモデルが allowlist 外に切り替わる場合はトグルを拒否するようになりました
- Opus 4.8 が有効になっていない組織で Fable 5 の auto モードが失敗する問題を修正 — classifier が利用可能な最良の Opus モデルにフォールバックするようになりました
- Read/Edit/Write ツールパスの hook `if` 条件を修正：`Edit(src/**)`, `Read(~/.ssh/**)`, `Read(.env)` などのドキュメント記載パターンが正しくマッチするようになりました
- `.claude/settings.json` が絶対パスターゲットへのシンボリックリンクの場合に Linux sandbox が起動しない問題を修正
- SSH 越しの tmux 内でシステムクリップボードに `/copy` とマウス選択コピーが届かない問題、および tmux 3.2 未満でペーストバッファが読み込まれない問題を修正
- Remote Control が web/モバイルから接続するとセッションのモデルが無言で切り替わる問題を修正
- Remote Control の切断通知が人間可読な理由ではなく数値コードを表示する問題、および接続失敗時に会話トランスクリプトに重複行が追加される問題を修正
- 別アカウントにサインインしても Remote Control セッションが切断されない問題を修正
- `/cd` とワークツリーの移動後もセッションが前のディレクトリの git ブランチを報告し続ける問題を修正
- `claude agents` で一つのウィンドウで戻るを押すと同じセッションにアタッチしている他のウィンドウが切断される問題を修正
- `/bg` でターンの途中でバックグラウンド化したセッションが継続すべき処理がない場合でも "Working" と表示し続ける問題を修正
- PR URL によるバックグラウンドエージェント検索を修正：スケジュール済みウェイクアップ中またはジョブがブロックされている間に作成された PR が `claude agents` 検索に表示されるようになりました
- Windows でエージェントビューの入力にテキストカーソルが表示されない問題を修正
- `claude --bg -cn <name>` でセッション名が設定されない問題を修正
- バックグラウンドセッションが再起動前に保存状態内の Windows ネットワークパスを無効化するよう修正
- バックグラウンドセッションの再起動が破損した状態ファイルからの不正な resume ID を拒否する問題を修正
- `~/.claude/daemon` ディレクトリに ReadOnly 属性が設定されている場合に Windows バックグラウンドサービスデーモンが起動しない問題を修正
- アイドル状態が長すぎた後にクラウドセッションが "Could not resolve authentication method" で失敗する問題を修正
- 自動更新をまたいで開いたままのウィンドウが返信を送信できない場合に明確なガイダンスを表示するよう改善、`claude daemon status` がバージョンスキューの動作を説明するようになりました

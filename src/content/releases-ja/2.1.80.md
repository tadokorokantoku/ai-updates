- Claude.ai レート制限使用量を表示するためのステータスラインスクリプト用 `rate_limits` フィールドを追加（5 時間と 7 日ウィンドウ、`used_percentage` と `resets_at` 付き）
- `source: 'settings'` プラグインマーケットプレイスソースを追加 — プラグインエントリを settings.json 内でインライン宣言
- ファイルパターンマッチングに加えて、プラグインヒントに CLI ツール使用量検出を追加
- 呼び出し時にモデル effort レベルをオーバーライドするスキルとスラッシュコマンド向けの `effort` frontmatter サポートを追加
- `--channels`（research preview）を追加 — MCP サーバーがセッションにメッセージをプッシュできるように
- 並列ツール呼び出しのある `--resume` が tool_use/tool_result ペアを `[Tool result missing]` プレースホルダーを表示する代わりにすべて復元するよう修正
- 非ブラウザ TLS フィンガープリントに対する Cloudflare bot 検出による voice モードの WebSocket 失敗を修正
- API プロキシ、Bedrock、または Vertex 経由の細粒度ツールストリーミング使用時の 400 エラーを修正
- 動作しない gateway とサードパーティプロバイダーのデプロイで `/remote-control` が表示される問題を修正
- `/sandbox` タブの切り替えが Tab や矢印キーに応答しない問題を修正
- 大きな git リポジトリでの `@` ファイル autocomplete の応答性を改善
- ステータスバーインジケータと一致するよう、auto が現在何に解決されているかを `/effort` が表示するよう改善
- `/permissions` を改善 — リスト内から Tab と矢印キーでタブを切り替え可能に
- バックグラウンドタスクパネルを改善 — 左矢印がリストビューから閉じる
- 2 段階フローの代わりに単一の `/plugin install` コマンドを使用するようプラグインインストールヒントをシンプル化
- 大きなリポジトリでの起動時メモリ使用量を削減（250k ファイルリポジトリで約 80 MB 節約）
- 前回のセッションから `remote-settings.json` がキャッシュされている際、管理対象設定（`enabledPlugins`、`permissions.defaultMode`、policy 設定された env 変数）が起動時に適用されない問題を修正

- ネイティブ Windows をサポート（Git for Windows が必要）
- 環境変数 AWS_BEARER_TOKEN_BEDROCK による Bedrock API キーをサポート
- Settings: /doctor で不正な設定ファイルを特定・修正可能に
- `--append-system-prompt` を --print/-p だけでなくインタラクティブモードでも利用可能に
- 自動コンパクトの警告閾値を 60% から 80% に引き上げ
- シェルスナップショットでスペースを含むユーザーディレクトリの処理を修正
- OTEL リソースに os.type、os.version、host.arch、wsl.version（WSL 上で実行時）を追加
- カスタムスラッシュコマンド: サブディレクトリ内のユーザーレベルコマンドを修正
- Plan モード: サブタスクから拒否されたプランが破棄される問題を修正

- `/config key=value` 構文を追加し、プロンプトから任意の設定を変更可能に（例：`/config thinking=false`）— インタラクティブモード、`-p`、Remote Control で動作
- `sandbox.allowAppleEvents` オプトイン設定を追加。macOS でサンドボックス化されたコマンドが Apple Events を送信できるようになる
- `CLAUDE_CLIENT_PRESENCE_FILE` 環境変数を追加：マーカーファイルを指定することで、機器の前にいる間のモバイルプッシュ通知を抑制
- バンドルされた Bun ランタイムを 1.4 にアップグレード
- 長い段落のストリーミングを改善：最初の改行を待たずに行単位でテキストが表示されるように
- 自動リトライを改善：thinking 中の API 接続断が「Connection closed while thinking」を表示する代わりに自動リトライされるように
- サブエージェントパネルを改善：アイドル状態のサブエージェントが 30 秒後に自動非表示、リストが最大 5 行に制限されスクロールヒントを表示、フッターにキーボードヒントを表示
- MCP OAuth ブラウザページを Claude Code のビジュアルスタイルに合わせ、成功時に自動クローズするように改善
- フルスクリーンモードでの URL 開封を Cmd+click（macOS）/ Ctrl+click を必須に変更し、ネイティブターミナルの動作に合わせた
- `Improved N memories` の行が verbose モード以外では個別ファイルを列挙しないように変更
- カスタム `ANTHROPIC_BASE_URL` および Foundry でリクエストごとの attestation トークンが毎ターン変わることでプロンプトキャッシュが読み取られない問題を修正
- ネットワークドライブやクラウド同期フォルダで Write/Edit が 0 バイトまたは切り詰められたファイルを生成する問題を修正
- Apple Events entitlement を追加することで、macOS で `open`、`osascript`、ブラウザベースの認証フローがエラー -600 で失敗する問題を修正
- 起動時の退行（2.1.169 で導入、クリーンな環境での起動時に約 120ms 増加）を修正：MCP サーバーが設定されていない場合、最初のプロンプトが managed-settings フェッチを待たないように
- 低速ネットワーク環境でアカウント設定のフェッチが遅い場合に最大 15 秒間ブランクターミナルで起動がブロックされる問題を修正
- `.claude.json` に破損した null プロジェクトエントリが含まれている場合の起動クラッシュ（`TypeError: Cannot read properties of null`）を修正
- Spotlight が再インデックス中のとき、macOS TUI がセッション開始時に Ctrl+C が反応しなくなってフリーズする問題を修正
- 別の Claude Code プロセスが 30 日のトランスクリプトクリーンアップを実行すると、長時間アイドル状態のセッションの履歴が失われる問題を修正
- フォアグラウンドのサブエージェントが無制限のネストチェーンを生成する問題を修正；バックグラウンドサブエージェントと同じ 5 レベルの深さ制限を適用するように
- モデル切り替え直後に `/recap` や会話フォークが以前のモデルを使用する問題を修正
- サブエージェントの「Thinking」時間が自身の経過時間ではなく親エージェントの経過時間を表示する問題を修正
- ネストされたエージェントを待機しているサブエージェントがエージェントパネルで「waiting」ではなくティック経過時間を表示する問題を修正
- API リトライインジケーター（「Retrying in 0s · attempt N/10」）がリトライ成功後も画面に残る問題を修正
- 残り有効期限が短い AWS `awsCredentialExport` 認証情報が毎分認証情報の更新を引き起こす問題を修正、`aws configure export-credentials` の JSON 形式も受け付けるように
- `claude mcp get`/`list` で tools/list が失敗しているときに `✓ Connected` と表示される問題を修正；`! Connected · tools fetch failed` とエラー詳細を表示するように
- `/remote-control` が古い「connecting…」の行を残す問題を修正；接続後にトランスクリプトで確認メッセージを表示するように
- Windows で bare `git` が解決できない場合に ExitWorktree がクリーンなワークツリーの削除を「Could not verify worktree state」で拒否する問題を修正
- `~/.claude` がシンボリックリンクされている配下で `~/.claude/settings.json` が相対シンボリックリンクの場合に `/effort` や `/model` などの設定変更が ENOENT で失敗する問題を修正
- IntelliJ および VS Code でコンテキストリマインダーの IDE 選択行番号が 1 つずれる問題を修正
- フルスクリーンで modifier+drag によるネイティブターミナル選択後に Ctrl+C がクリップボードをアプリの前の選択内容で上書きする問題を修正
- クリップボードにテキストが含まれている場合に Ctrl+V が「No image found in clipboard」を表示してペーストできない問題を修正
- エージェントディレクトリが既に存在する場合にエージェント作成が「EEXIST: file already exists」で失敗する問題を修正（Windows/OneDrive）
- AskUserQuestion のプレビューコンテンツがワードラップされずダイアログの端で切れる問題を修正
- AskUserQuestion のマルチセレクト質問で送信時に入力された「Other」の自由回答が消える問題を修正
- `/stats` の「Most active day」および日次トークンチャートの日付が UTC-negative タイムゾーンで 1 日早く表示される問題を修正
- `/copy` および Linux でのコピーオンセレクトが Claude Code 起動後にインストールされたクリップボードユーティリティを検出しない問題を修正
- Write（ファイル作成）プレビューでタブインデントのコードが誤ったインデントで表示される問題を修正
- ターン中にキューに入れられたユーザープロンプトがトランスクリプトで全幅の背景ハイライトを表示しない問題を修正
- Ghostty でアクティビティスピナーのパルスが誤ったグリフサイズで表示される問題を修正

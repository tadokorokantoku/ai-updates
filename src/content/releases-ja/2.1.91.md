- `_meta["anthropic/maxResultSizeChars"]` アノテーション（最大 500K）経由で MCP ツール結果の永続化オーバーライドを追加。DB スキーマなどのより大きな結果を切り詰めずに通過可能に
- スキル、カスタムスラッシュコマンド、プラグインコマンドでのインラインシェル実行を無効化する `disableSkillShellExecution` 設定を追加
- `claude-cli://open?q=` ディープリンクの複数行プロンプトサポートを追加（エンコードされた改行 `%0A` は拒否されない）
- プラグインが `bin/` 配下に実行可能ファイルを同梱し、Bash ツールから bare コマンドとして呼び出せるように
- async トランスクリプト書き込みが無言で失敗したときに会話履歴を失う可能性がある `--resume` でのトランスクリプトチェーンの切断を修正
- iTerm2、kitty、WezTerm、Ghostty、Windows Terminal で `cmd+delete` が行頭まで削除しない問題を修正
- プランモードのリモートセッションでコンテナ再起動後にプランファイルの追跡を失い、プラン編集で権限プロンプトと空のプラン承認モーダルを引き起こす問題を修正
- settings.json の `permissions.defaultMode: "auto"` の JSON スキーマ検証を修正
- Windows バージョンクリーンアップがアクティブなバージョンの rollback コピーを保護しない問題を修正
- スラッシュメニューから消える代わりに、`/feedback` が利用できない理由を説明するように
- ツール表面の決定、コンテキスト管理、キャッシング戦略を含むエージェント設計パターンのための `/claude-api` スキルガイダンスを改善
- `Bun.stripANSI` 経由でルーティングして Bun での `stripAnsi` を高速化しパフォーマンスを改善
- Edit ツールがより短い `old_string` アンカーを使用するように。出力トークンを削減

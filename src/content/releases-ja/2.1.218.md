- `/code-review` をバックグラウンドサブエージェントとして実行するように変更。レビュー作業が会話を埋め尽くさなくなり、積み重ねられた slash command をそのままレビュー対象として保持する
- `--ax-screen-reader` モードで、単語・行削除（`Option+Delete`、`Ctrl+W`、`Cmd+Backspace`、`Ctrl+U`、`Ctrl+K`）時に削除されたテキストをスクリーンリーダーでアナウンスするように追加
- `C:\Users\unicorn` のような `\u` プレフィックス付きセグメントを含む Windows パスが、tool の入力で CJK 文字に壊れてそのファイルにアクセスできなくなっていた不具合を修正
- 左矢印キーで会話が undo なしに破棄されていた不具合を修正: 編集直後の押下は確認を求めるようになり、agent view での Esc はバックグラウンドに送った会話に戻るように
- サーバーへの接続に失敗した際、`claude mcp list` と `/mcp` に HTTP ステータスとエラー文言を追加。また MCP config の値に前後の見えない空白がある場合の警告を追加
- ペーストされた改行を Ctrl+J としてエンコードするターミナルで、複数行ペーストが `j` に置き換わって1行に潰れていた不具合を修正
- メッセージピッカーから compact した後、`/context` が compact 前の古いトークン使用量を報告していた不具合を修正
- "review my auth changes" のような説明的な引数で `/ultrareview` が失敗していた不具合を修正 — 現在のブランチのレビューを実行し、そのテキストを所見への注記として適用するように
- 非対話セッションで `/code-review ultra` が黙ってローカルレビューを実行していた不具合を修正 — クラウドレビューを起動するように
- Bedrock の application-inference-profile ARN や他の config でマッピングされた upstream model ID を、設定されたモデルの料金で価格計算するようゲートウェイの spend metering を修正
- 長い IDE 選択範囲が絵文字の途中で切り詰められた際の文字化け、および tool executor のエラーが黙って握りつぶされることがあった不具合を修正
- phantom turn を開始して放置してしまうことがあった engine teardown のレースコンディションを修正。close 後にプッシュされた入力は一貫して拒否されるように
- tool 呼び出しが中断された後に "[Request interrupted by user]" という誤ったメッセージが表示されていた不具合、および tool がレスポンス途中で中断した際に対になっていない `tool_use` ブロックが transcript に残っていた不具合を修正
- `--ax-screen-reader` モードで、入力欄の末尾に入力されたスペースを VoiceOver がエコーする代わりに "new line" と読み上げていた不具合を修正
- plugin パネルと settings パネルがターミナルカーソルをフォーカス行に移動させておらず、スクリーンリーダーや拡大鏡が矢印キーでの操作に追従できなかった不具合を修正
- 深くネストされた監視対象ディレクトリツリーが削除・移動された場合や、深くネストされた UI ツリーを描画する場合のクラッシュ（maximum call stack exceeded）を修正
- セッションが PR を作成・リンクした直後に終了した際、pull request イベントが失われることがあった不具合を修正
- 分割された AWS リージョンや proxy 専用ネットワークで、Bedrock のセットアップウィザードが assume-role プロファイルの検証に失敗していた不具合を修正
- システムクロックの調整後に turn の所要時間が稀に負の値や不正な値になっていた不具合を修正。turn の計測を単調クロックで行うように
- "N MCP servers need authentication" という起動時の通知が、claude.ai で接続されていない claude.ai コネクタを過剰にカウントしていた不具合を修正
- 履歴の書き込みが競合または失敗した際に、プロンプト履歴のエントリが失われたり重複したりしていた不具合を修正
- 大きな thinking budget でコンテキストオーバーフローエラーが発生した後、同一の失敗確定リクエストを再送し続けるリトライループを修正。`Ctrl+B` によるバックグラウンド化も他の経路と同じバックグラウンドシェルの上限を適用するように
- 信頼されていないフォルダから agent frontmatter の hooks が実行されていた不具合を修正: hooks の実行には agent ファイル自身のフォルダがワークスペース信頼を承認済みであることが必要に
- headless セッションおよび SDK セッションで、compaction 後に fork-session の lineage が失われていた不具合を修正
- 履歴に不正な delta attachment が含まれる場合に、resume したセッションが毎ターン失敗したり resume 時にクラッシュしたりしていた不具合を修正

- auto mode を改善: 危険な rm、バックグラウンドの `&`、疑わしい Windows パスのチェックで permission ダイアログを開かなくなり、auto-mode の分類器がこれらを判定するように

- `/config model=<x>` または Remote Control でのモデル切り替えの結果として fast mode が変化した際にアナウンスを追加

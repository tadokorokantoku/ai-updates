- コマンド検索時のあいまい検索結果を改善
- VS Code 拡張で UI 全体を通して `chat.fontSize` と `chat.fontFamily` 設定を尊重するよう改善。リロードなしでフォント変更を即座に反映
- 指定したアイドル時間後に SDK モードを自動終了する `CLAUDE_CODE_EXIT_AFTER_STOP_DELAY` 環境変数を追加。自動化ワークフローやスクリプトに便利
- `ignorePatterns` をプロジェクト設定から localSettings の deny permissions に移行
- 空文字列や falsy 値を持つ項目 (例: `/hooks` メニュー) でメニューナビゲーションが止まる問題を修正
